Description

Object type: plaque / stela
Material: limestone
Type of Inscription: epitaph
Execution Technique: carved
Letters: yes
Provenance: Napoca
Findspot: Zigeunergasse, Romania
Date: due to the fact that Napoca is a municipium, during the period Hadrian-Marcus Aureliu / Commodus

Bibliography:

  • CIL III 7664
  • ILD 564

Text

[---]
[vix(it) ann(is)] XXXV Aelia Proba vix(it) an[n(is)..]
[m(ensibus).. d(iebus) ?] XI P(ublius) Aelius Ingenu(u) s vix(it) [ann(is)..]
[P(ublius) Ae]lius Probus flamem(sic!) muni[cipii]
5[---] pientissimis et sibi viv[o fecit].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

...lived 35 years...AeliaProba lived...and 11 days...P.Aelius Ingenuus lived...(Publius) Aelius Probus, flamen of the municipium, dedicated the monument to his very pious...and for himself, while still alive.
