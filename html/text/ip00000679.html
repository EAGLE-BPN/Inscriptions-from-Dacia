Text

C(o)h(ortis) III.

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la cohors III.


Commentary:

J. Szilágyi: c(ohors) Hi(spanorum)?; N. Gudea; coh(ors) III (Dacorum), représentant le type I/2 des etampilles de cette troupe; I. Piso: coh(ortis) III (Campestris), lecture violemment rejetée par N. Gudea.
