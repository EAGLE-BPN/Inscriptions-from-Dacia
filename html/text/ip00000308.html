Description

Object type: Dé d'une base de monument funéraire
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: hederae et points de séparation
Provenance: Certia
Findspot: Romita, Roumanie
Date: Il est difficile de dater l’inscription sur la base de l’écriture. Le A, le P et le R sont assez étroits, mais le le O est parfaitement rond. Nous nous trouvons probablement dans l’époque des Sévères.

Text

D(is) M(anibus)
C(ai) Campani(i) Vita
lis (centurionis) coh(ortis) I Bat(avorum)
(miliariae) stip(endiorum) VIIII
55 vix(it) an(nis) XXVII
Florius Viri
lis vet(eranus) ex dec(urione)
nepoti p(ientissimo) p(onendum) c(uravit).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

Aux dieux Mânes de Caius Campanius Vitalis, centurion de la cohors I Batavorum miliaria, après un service militaire de 9 ans ; a vécu 27 ans. Florius Virilis, vétéran, ancien décurion, pris le soin d’ériger (ceci) au neveu très pieux.


Commentary:

Le nom du défunt se trouve au génitif. Pour le nomen Campanius voir W. Schulze, Eigennamen, p. 352, 525, 532 ; H. Solin, O. Salomies, Repertorium, p. 44 ; pour Florius voir W. Schulze, Eigennamen, p. 480 ; H. Solin, O. Salomies, Repertorium, p. 80. Voir pour la troupe de Bataves J. E. H. Spaul, Cohors2, p. 88-90 ; C. C. Petolescu, Auxilia, p. 83-84. La charge de décurion de Florius Virilis prouve que la cohorte était equitata.
