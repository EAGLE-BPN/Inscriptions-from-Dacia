Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Samum
Findspot: Cășei, Roumanie
Date: Probablement à partir de Sévère Alexandre, comme toutes les autres inscriptions de beneficiarii consularis de Samum.

Bibliography:

Text

I(ovi) o(ptimo) m(aximo)
M(arcus) Aurel(ius)
Moenenus
[b(ene)f(iciarius)] co(n)s(ularis) leg(ionis) V [M(acedonicae)] pi(a)e
55 a{u}g[ens sub sign]is r[eg(ione) ?]
Ans(amensium ?) pro sal(ute)
sua suorumq(ue)
omnium v(otum) l(ibens)
m(erito) [s(olvit) ?].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

À Jupiter très bon (et) très grand. Marcus Aurelius Moenenus, (bénéficiaire) du consulaire, provenant de la legio V (Macedonica) la pieuse( ?), se trouvant en mission sous les étendards dans la regio Ansamensium( ?), (s’est acquitté) de (son) vœu de bon gré (et) à bon droit pour sa santé et pour celle de tous les siens.


Commentary:

La lecture d’en haut, avec beacoup de signes d’interrogation, d’après C. Daicoviciu ; voir aussi D. Isac. Pour la regio Ansamensium voir AE 1957, 326. Moenenus a une résonance grecque, mais le nom a été probablement mal copié ; il s’agit peut-être de Monimus (Μόνιμος) ; voir W. Pape, Wörterbuch II, p. 943.
