Description

Object type: Pièce d'armure d'un légionnaire
Material: bronze
Type of Inscription: Inscription sur une armure romaine
Execution Technique: en pointillé
Letters: points de séparation 0,40,7
Provenance: Potaissa
Findspot: Turda/Torda, Roumanie
Date: À partir de 168, quand la legio V Macedonica est arrivée en Dacie.

Bibliography:

Text

(Centuria) Faustini Suc(cessius) Resp(ectus).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Successius Respectus de la centurie de Faustinus.


Commentary:

Garniture d’armure en bronze, décorée au repoussé et partiellement argentée, trois fragments jointifs (17,2 x 9,2 cm). Dans le registre supérieur est représenté un aigle, dans le registre médian un buste de Minerve et dans le registre inférieur deux boucliers croisés. Sur la marge droite on a écrit un texte en pointillés. Bărbulescu (2012) a bien vu les lettres, mais a supposé un nom Suc(cessus) Resp(ectus), composé de deux cognomina. Il s’agit probablement d’un nomen comme Successius, dérivé d’un Successus, selon la mode des provinces celtiques occidentales; voir AIJ 329 (Poetovio) et CIL III 11532 = AE 2007, 1077 (Virunum).
