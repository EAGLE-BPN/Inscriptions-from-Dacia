Itụ(raeorum)

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant aux Ituraei?


Commentary:

Les deux exemplaires illustrés par Gudea ont conservé le début d’une seconde haste du V. Le même auteur, suivi par tous les autres, croit qu’il s’agit de la cohors I Ituraeorum sagittariorum, ce qui est possible, mais assez incertain. Les Ituraei sont une population sémite du nord-est de la Palestine.
