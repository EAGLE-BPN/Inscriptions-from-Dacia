Text

D(is) [M(anibus)]
Ostilia [......]
vix(it) a[nn(is) ...]
Ostilị[a ......]
55 ta ṿ[ix(it) n(is) ..
- - - - - -].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Aux dieux (Mânes). Ostilia ..., a vécu ... (ans). Ostilia ...ta, a vécu (... ans). ....


Commentary:

Malgré le fait que dans le médaillon sont représentés une femme et un homme, le second défunt est toujours un personnage féminin, car dans la l. 5 les lettres TA représentent la fin d’un cognomen féminin. La forme habituelle du nomen est Hostilia ; voir W. Schulze, Eigennamen, p. 30, 175, 442, n. 4, 456 ; H. Solin, O. Salomies, Repertorium, p. 94 ; pour la forme Ostilius voir Repertorium, p. 134.
