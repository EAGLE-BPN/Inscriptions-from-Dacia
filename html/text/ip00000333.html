Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: Vers le milieu du IIIème siècle, en raison de la forme du A, avec la barre en angle.

Bibliography:

  • I. Bajusz, AMP 16, 1992, p. 149, 174, fig. 17 (dessin)
  • idem, AMP 25, 2003, p. 166, no 1, 183, pl. V (dessin)
  • AE 2003, 1468
  • ILD 692.

Text

Neme
si sac(rum)
Nepos
te(sserarius) n(umeri) Pal(myrenorum).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Consacré à Nemesis. Nepos, tesserarius du numerus Palmyrenorum, (érigea ceci).


Commentary:

Pour le tesserarius (porteur de tessera, celui qui transmettait les ordres) des troupes auxiliaires voir A. v. Domaszewski, Rangordnung2, p. 58-59, 62. Le dédicant fait mention seulement de son cognomen.
