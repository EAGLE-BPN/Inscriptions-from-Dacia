Description

Object type: Autel votif ou base de statue
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Du règne de Caracalla.

Bibliography:

  • CIL III 884
  • A. Buonopane, V. La Monaca, dans : Epigrafi romane, p. 296, no 43(dessin).

Text

I(ovi) o(ptimo) m(aximo)
p(ro) s(alute) d(omini) n(ostri)
M(arci) Aur(elii) Anto
nini Imp(eratoris)
5Aurel(ius) Ru
sticus c(ustos) a [ r(morum) ?]
posuit.


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon(et) très grand pour la santé de notre seigneur, l’empereur Marcus Aurelius Antoninus. Aurelius Rusticus, gardien des armes, érigea(ceci).


Commentary:

Bien que les épithètes p(ius) f(elix) manquent, l’appellatif d(omini) n(ostri) et le nomen Aurelius, pas accompagné par praenomen, du dédicant parlent en faveur de Caracalla, plutôt que de Marc Aurèle. Il faut peut-être partir de l’idée qu’une dédicace faite à Jupiter pour le salut de l’empereur devait être faite par une personne qui revêtait une charge. Même en supposant une erreur de lapicide ou de lecture, la meilleure solution semble être c(ustos) a [ r(morum)].
