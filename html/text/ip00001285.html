Text

I(ovi) O(ptimo) M(aximo)
M(arcus) Cocc(e)
ius
Themo
55 v(otum) s(olvit) l(ibens) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon (et) très grand. Marcus Cocceius Themo s'est acquitté de (son) voeu de bon gré (et) à bon droit.


Commentary:

Le cognomen du dédicant, Themo est palmyrénien, en tout cas sémite.
