Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Brique fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 1,21,5
Provenance: ?
Findspot: Bologa, Roumanie
Date: À partir du règne d’Hadrien la cohors I Aelia Gaesatorum semble partager le camp de Bologa avec la cohors II Hispanorum.

Text

C(o)h(ortis) I Ae(liae) G(aesatorum).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la cohors I Aelia Gaesatorum.


Commentary:

Estampille en négatif et lettres imprimées. Le signaculum, qui peut à la rigueur être interprété comme cartouche, était de forme aproximativement rectangulaire, avec les marges légèrement arrondies; il peut être distingué dans la mesure où on le pressait. C. Torma: C(o)h(ortis) I Aeg(yptiorum); A. v. Domaszewski (CIL): C(o)h(ors) I G(allorum); c’est I. Szilágyi qui a reconnu la troupe des Gaesati; voir pour cette troupe C. C. Petolescu, Auxilia Daciae, p. 103-104.
