Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: À partir de l'arrivée de la légion à Potaissa.

Text

I(ovi) o(ptimo) m(aximo)
Do(licheno) Ti(berius)
T(- - - ?) Pat(ernus)b(ene)f(iciarius) leg(ati)
v(otum) s(olvit) l(ibens) m(erito).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon(et) très grand, Dolichenus. Tiberius T(…) Paternus(?), beneficiarius du légat(de la légion), s’est acquitté de(son) vœu de bon gré(et) à bon droit.
