Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: andésite
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Micia
Findspot: Vețel, Roumanie
Date: Du IIIe siècle, en raison de l'épithète Deus portée par Mercure.

Bibliography:

  • G. Téglás, AEM 9, 1885, p. 239, no 6
  • idem, ErdMúz 5, 1888, p. 241
  • CIL III 7856
  • R. Münsterberg, J. Oehler, JÖAI 5, 1902, Bbl. 120

Text

Deo Mercuri
o Marcus Anto
nius Sabini
anus princeps
5v(otum) s(olvit) l(ibens) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Au dieu Mercure. Marcus Antonius Sabinianus, princeps, s'est acquitté de (son) voeu de bon gré (et) à bon droit.


Commentary:

Il est difficile à décider si le dédicant était princeps d'une ethnie ou centurio princeps.
