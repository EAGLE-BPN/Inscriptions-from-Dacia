Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle ou plaque funéraire,
Material: tuf volcanique
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: Les archers Palmyréniens ont été organisés en numeri vers la fin du règne d’Antonin le Pieux ; en raison du nomen Aurelius, l’inscription peut être datée à partir du règne de Marc Aurèle.

Text

[ [. ] [ ] [ ]]
[ [. ] [] A]ur(elius) G[ [. ] []]
[ [. ] []]rio m [ il(es)]
[ n(umeri) Palm(yrenorum)] Poro [ l(issensium)]
5[ ..] p(osuit).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

… Aurelius …rio, militaire du(numerus Palmyrenorum) Porolissensium érigea(ceci) à ….


Commentary:

Wollmann, Ardevan : [ - - -] Ro [ cius / Satu]rio m(iles) [ num(eri) / Palm(yrenorum)] Poro [ liss(ensium)] / [ p(ientissimae)] p(osuit)]. Les deux auteurs virent au début de la l. 1 une hedera, ce qui ne se confirme pas ; on y voit un A. Dans la même ligne, après le R, pourrait être un petit signe de séparation, suivi par un signe qui ressemble à un G. Pour le cognomen du personnage, on peut consulter une longue liste de cognomina finissant en –rio chez H. Solin, O. Salomies, Repertorium, p. 439. Le numerus Palmyrenorum Porolissensium a sa garnison à Porolissum.
