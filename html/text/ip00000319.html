Author: Ioan Piso

Description

Object type: Plaque honorifique encastrée dans une base
Material: calcaire
Type of Inscription: Inscription honorifique de Caracalla
Execution Technique: sculptées
Letters: Points de séparation 3.5
Provenance:
Findspot: Buciumi, Roumanie
Date: Ccomme l’épithète triomphale Germanicus maximus manque, l’inscription est datable des années 212-213 ; voir D. Kienast, Kaisertabelle, p. 162-163. L’inscription semble avoir un rapport avec les préparatifs en vue de la visite de Caracalla en Dacie.

Bibliography:

Text

Imp(erator) Caes(ar) M(arcus) [Aurel(ius)]
Anton(inus) pius [felix]
Aug(ustus) Part(hicus) max(imus) [Brit(annicus) ma]
xim(us) pontif(ex) m[ax(imus) coh(ors)]
55 II Nerv(ia) Britt(onum) Ạ[nto]
niniana p(ia) f(idelis) statu[am pos(uit)].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

L’empereur César Marc (Aurèle) Antonin, pieux (et heureux) Auguste, grand vainqueur des Parthes, grand (vainqueur des Bretons), grand pontife. la (cohors) II Nervia Brittonum Antoniniana pieuse (et) fidèle, (érigea) la statue.


Commentary:

Le caractère du monument n’et pas très clair. Même si le nom de l’empereur se trouve au nominatif et pas au datif, l’explication la plus plausible reste que nous avons affaire à une statue impériale. Pourtant, la largeur totale de la plaque ne dépassait pas 75, ce qui est peu pour une telle statue. Il n’est donc pas exclu qu’il s’agisse d’une statue d’une divinité quelconque, érigée pour le salut de Caracalla. Pour la cohors II Nervia Brittonum ∞ voir J. E. H. Spaul, Cohors2, p. 201; C. C. Petolescu, Auxilia, p. 90-91.
