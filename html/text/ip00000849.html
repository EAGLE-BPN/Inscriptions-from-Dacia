Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Base de statue fragmentaire
Material: calcaire
Type of Inscription: Inscription honorifique
Execution Technique: sculptées
Letters:
Provenance: Resculum?
Findspot: Bologa, Roumanie
Date: À partir de 212, en raison de la formul finale.

Bibliography:

  • N. Gudea, AMN 9, 1972, p. 418, no 33, 427, fig. 6/4 (photo)
  • I. Piso, ms.

Text

[- - - - - -
numi]ni [maiesta
tique eius].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

…, à (sa) force divine (et à sa majesté).


Commentary:

Il s’agit de la formule finale d’une inscription impériale sur une base de statue.
