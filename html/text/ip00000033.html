Description

Object type: Autel votif ou base de statue
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Il n'y a pas des critères pour la datation.

Bibliography:

  • CIL III 890
  • I. I. Russu, AMN, 1968, p. 466-467, no 12, fig 12
  • AE 1971, 363
  • ILD 471.

Text

[ I(ovi)] o(ptimo) m(aximo)
ẹt S(ilvano ?)
P(ublius) S(- - -) C(- - -).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À(Jupiter) très bon(et) très grand et à Silvain(?). Publius S… C….


Commentary:

Le dédicant a utilisé les initiales de ses tria nomina. Il était très connu dans la zone ou bien il a érigé le monument chez soi-même.
