Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle funéraire,
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: hederae
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Du IIe siècle, en raison du nom de facture pérégrine du défunt.

Bibliography:

  • I. Téglás, AÉrt 2, 191, p. 48(dessin)
  • I. I. Russu, AISC 4, 1941-1943(1944), p. 231, no 33
  • ILD 22.

Text

[ [. ] [ ] [ ]]
Sur [ u]s [ ….]
omari [ f(ilius)?vix(it)]
[ a]ṇ(nis) ḶX E(?)
5[ [. ] [ ] [ ]].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

…, Surus, fils de …omarus,(a vécu) 60 ans. ….


Commentary:

[ - - -]omarus est sans doute un nom celtique(Russu). Dans ce contexte Surus est tout aussi bien celtique.
