Text

M(arcus) Opellius Maximus
[dec(urio) Mo]ntanensium Herculi
[pro sal(ute) Ma]ḷvensium ex voto posuit.


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Marcus Opellius Maximus, décurion des Montanenses, érigea (ceci) à Hercule pour le salut des Montanenses.


Commentary:

C. S. Nicolăescu-Plopșor: [Mal]vensium; C. Daicoviciu: [Aeq]uensium; D. Tudor: [Aq]uensium. Correcte est la première lecture, ce qui ne signifie pas que Cioroiul Nou pourrait être identifiée à l'ancienne Malva. Les Montanenses étaient les habitants de Montana, de Mésie Inférieure.
