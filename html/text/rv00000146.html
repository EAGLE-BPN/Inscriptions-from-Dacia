Source: UNIVERSITATEA BABES BOLYAI


Author: Rada Varga

Description

Object type: altar / statue base
Type of Inscription: votive inscription
Execution Technique: carved
Letters: no 4,5
Provenance: Napoca
Findspot: , Romania
Date: due to Augg[[g(ustorum)]], in 209-211 or 198-211, due to the career of the character, around 200.

Bibliography:

  • AE 1934, 14
  • AE 1977, 700

Text

I(ovi) o(ptimo) m(aximo) Dolic(heno)
pro salute
Ael(ii) Lycini
proc(uratoris) Augg⟦g(ustorum)⟧
5[[---][ ]].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

To Iupiter the best and the greatest, Dolichenus, for the health of Aelius Lycinus, procurator of the 3 emperors
