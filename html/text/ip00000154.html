Description

Object type: Plaque funéraire

Text

D(is) M(anibus)
Aia Nandonis vixit
annis LXXX Andrada
Bituvantis vix(it) an(n)is
5LXXX Bricena vixit an(n)is
XL Iusta vixit an(n)is XXX
Bedarus vixit(annis) XII pos(t) obi
tum ei(us) Herculanus liber
tus patron(a)e bene mer(enti).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Aux dieux Mânes. Aia, fille de Nando, a vécu 80 ans ; Andrada, fille de Bituvas, a vécu 80 ans ; Bricena a vécu 40 ans ; Iusta a vécu 30 ans ; Bedarus a vécu 12(ans) ; après sa mort(de Aia Nandonis), Herculanus, en tant qu’affranchi,(érigea ceci) à la patronne qui a bien mérité.


Commentary:

Apparemment, dans l'onomastique des personnages il y a des éléments illyriens, celtiques et thraces. Nous avons pourtant affaire à un famille illyrienne de statut pérégrin.
