Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Plaque de construction?
Material: calcaire
Type of Inscription: Inscription de construction?
Execution Technique: sculptées
Letters: 12
Provenance: Certia
Findspot: Romita, Roumanie
Date: Il s’agit plutôt d’une inscription impériale de construction. Les empereurs les plus significatifs portant dans leur titulature la seconde acclamation impériale sont Hadrien, à partir de 135 (D. Kienast, Kaisertabelle 130) et, surtout, Antonin le Pieux, à partir de 142 (D. Kienast, p. 135). Probablement nous avons affaire au second empereur, ce qui pourrait constituer un indice pour la transformation en pierre du camp de Romita.

Bibliography:

  • N. Gudea, V. Lucăcel, Inscripții, p. 27, no 52 (photo)
  • ILD 654

Text

[[. ] [ ] [ ]]
tribuṇ(icia) [potest(ate) ….]
im[p(erator)] II [co(n)s(ul) …. p(ater) p(atriae) fecit ?].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

(L’empereur …), investi avec (la puissance) tribunicienne (pour la … fois), acclamé imperator deux fois, (consul …, père de la patrie, a construit ceci?).


Commentary:

Il est difficile de supposer une dédicace avec des lettres de 12 faite par un tribunus militum. Il s’agit plutôt d’une inscription impériale de construction. Pour le camp de Certia voir A. V. Matei, I. Bajusz, Romita, p. 20-67 ; F. Marcu, Roman Forts, p. 101-114.
