Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Il n'y a pas des critères pour la datation.

Bibliography:

  • I, AÉrt 24, 1904, p. 21

Text

I(ovi) o(ptimo) m(aximo)
Dius
pos(u)i [ t?].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon(et) très grand. Dius érigea(ceci).


Commentary:

On constate la forme populaire posit ou bien posi, à la Ière personne. Le dédicant porte un nom grec, Δῖος(Δεῖος), étant de condition sociale modeste, probablement esclave.
