Description

Object type: Base de monument funéraire
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: 7,5
Provenance: Porolissum?
Findspot: Moigrad?, Roumanie
Date: Probablement à partir de Caracalla, en raison du nomen Aurelius sans praenomen.

Bibliography:

  • I. Piso, D. Deac, R. Zăgreanu, AMP 35, 2015.

Text

D(is) [M(anibus)]
A[u]r(elius) Dexṭ[er …
- - - - - -].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Aux dieux [Mânes]. Aurelius Dexter ….


Commentary:

Un nom comme Ardeṿ[anus] n’est pas connu à l’époque romaine. On remarque aussi que la distance entre le A et le R est un peu plus grande qu’entre les autres lettres. On peut donc compter avec une ligature entre A et un V plus petit. Le E est suivi d’une trace qui ne peut appartenir qu’à un V ou à un X. Des cognomina comme Devotus ne sont pas connu ; voir H. Solin, O. Salomies, Repertorium, p. 323. Il faut donc se décider pour Dexter, qui est particulièrement fréquent (ibidem).
