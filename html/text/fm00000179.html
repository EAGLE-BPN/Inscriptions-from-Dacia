Description

Object type: Altar
Material: Limestone
Type of Inscription: Funerary
Letters: No 4.5
Provenance: Capidava
Findspot: Reused in the Late Roman fort, Romania
Date: Second century AD, after 120 AD

Bibliography:

  • Gr. Florescu, Dacia 11-12, 1945-1947, p. 217
  • AE 1950, 76

Text

D(is) M(anibus) S(acrum)
Fabricia Q(uinti) f(ilia)
Saturnina p(i)a
vix(it) n(is) XXI
5C(aius) Munatius Venustus
praef(ectus) coh(ortis) I Ger(manorum)
coniugi piis-
simae pos(uit).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Consecrated to the Manes Gods! Fabricia, Quintus' daughter, Saturnina, pious, lived 21 years. Caius Munatius Venustus, prefect of the first Germanic cohort, set for his most pious wife.
