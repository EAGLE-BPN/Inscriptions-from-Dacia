Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: De 218-222, en raison de l'épithète impériale martelée.

Bibliography:

  • CIL III 902.

Text

Deae Neme
si Ulpius
Valerius
mil(es) leg(ionis)
5V M(acedonicae) p(iae) [ [ Ant(oninianae)]]
v(otum) s(olvit) l(ibens) m(erito).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À la déesse Nemesis. Ulpius Valerius, militaire de la Ve légion Macedonica, la pieuse, Antoniniana, s’est acquitté de(son) vœu de bon gré(et) à bon droit.


Commentary:

L. 3 : le nomen Valerius joue ici le rôle d’un cognomen ; l. : l’épithète impériale martelée ne peut appartenir qu’à Élagabal(Mommsen).
