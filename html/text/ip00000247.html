File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

…, à l’étranger(?). … érigèrent(ceci).


Commentary:

Russu, Milea : [ conter?]raneo / [ b(ene) m(erenti)] posuerunt. Extraneus est une notion appartenant au droit funéraire et se rapporte à toute personne qui n’a pas droit à une certaine enceinte funéraire ; voir CIL III 933 = ILJug 3, 2469 ; CIL III 96 = ILJug 3, 2400 ; CIL III 13124 = ILS 822(Salonae) ; exempli gratia [ non licet ext]raneo - - -. Pourtant, en jugeant d’après la photo d’assez mauvaise qualité, le R n’est pas tout-à-fait sûr. On peut y avoir aussi un nom comme Herculaneus.
