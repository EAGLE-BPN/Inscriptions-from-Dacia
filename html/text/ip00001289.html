Author: Ioan Piso

Description

Object type: Stèle funéraire, fragment
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: Points de séparation; hederae
Provenance: Aquae
Findspot: Valea Sângeorgiului, Roumanie
Date: Du IIe siècle, en raison du style de la stèle et de l'écriture.

Bibliography:

  • G. Téglás, AEM 9, 1886, p. 239, no 5
  • idem, ErdMúz 1888, p. 239
  • CIL III 7894
  • IDR III/3, 20.

Text

D(is) M(anibus)
Iu(lius) Maxi
mus vix(it)
an(nis) XXV
55 He(rennia?) Valentina
c(oniugi) c(arissimo).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Aux dieux Mânes. Iulius Maximus a vécu 25 ans. Herennia Valentina (érigea ceci) au très cher époux.


Commentary:

Manque le registre supérieur, qui contenait un médaillon avec le buste du défunt.
