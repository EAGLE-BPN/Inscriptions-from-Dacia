Description

Object type: Autel votif ou base de statue
Material: calcaire?
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: Points de séparation; hederae
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: En principe, de la période d’après la réforme de 173-175, qui apporta à la tête du portorium publicum per Illyricum un procurateur ; voir I. Piso, Fasti II, p. 297-298. D’autre part, indique la formule Aug. n. plutôt le IIIème siècle.

Text

[For]tunae
[Au]gustae
[Fe]lix Aug(usti)
n(ostri) vil(icus) sac
55 rumv(otum) s(olvit) l(ibens) m(erito).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Consacré à la Fortune Auguste. Felix, l’intendant de notre empereur s’est acquitté de (son) vœu de bon gré (et) à bon droit.


Commentary:

Felix est esclave impérial à la station douanière (statio portorii) de Porolissum ; voir pour celle-ci N. Gudea, AMP 12, 1988, p. 178–179, 189; ders., Vama, p. 277–278.
