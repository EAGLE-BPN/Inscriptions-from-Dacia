Source: UNIVERSITATEA BABES BOLYAI


Author: Rada Varga

Description

Object type: architrave
Material: marble
Type of Inscription: votive inscription
Execution Technique: carved
Letters: no
Provenance: Histria
Findspot: SE area and in the city wall, Romania
Date: 117-138 AD

Bibliography:

  • ISM I 147

Text

[Imp(eratori) Caes(ari) divi Traiani Part]hici filio, divi [Nervae nepoti Traiano Hadriano Aug(usto)Αὐτοκράτορι Καίσαρι θεοῦ Τραιανο]ῦ Παρθικοῦ υἱῷ, θε̣[οῦ Νέρουα υἱωνῷ, Τραιανῷ Ἁδριανῷ — — —]

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Emperor Caesar Traianus Hadrianus the Glorious, son of divus Traianus, conqueror of the Parthians, nephew of Divus Nerva…
