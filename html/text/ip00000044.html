Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Du règne de Gordien III, en raison de l'épithète portée par la légion.

Text

Invic(t)o
Aur(elius) Mon
tanus mil(es)
leg(ionis) V M(acedonicae) [ G]or(dianae)
5l(ibens) p(osuit).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Au(dieu) invaincu. Aurelius Montanus, militaire de la Ve légion Macedonica Gordiana, érigea(ceci) de bon gré.


Commentary:

Il s’agit plutôt d’un Baal syrien que de Mithra(Sanie).
