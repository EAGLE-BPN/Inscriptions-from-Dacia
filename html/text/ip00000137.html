Description

Object type: Plaque ou base de monument funéraire
Material: tuf volcanique?
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: Probablement à partir de la fin du IIème siècle, en raison de CIL III 80.

Text

Iul(ia) [ M]ansueta vix(it)
an(nis) [ .]XIII Val(erius) Valerian
[ us] vet(eranus) ex de [ c(urione) alae]
[ Frontonian]ae
5[ [. ] [ ] [ ]].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Iulia Mansueta a vécu … ans. Valerius Valerianus, vétéran, ancien décurion de l’(ala) Frontoniana, ….


Commentary:

Le cognomen Mansueta est répandu dans les provinces celtiques. Un Valerius Valerianus apparaît dans CIL III 80. Il est à noter que Iulia Mansueta porte un nomen différent de celui de Valerius Valerianus. Il est donc difficile d’établir le rapport de parentèle entre les deux, vu que la femme de Valerianus s’appelait Bonosa(CIL III 80). L. 2-3 : division erronée en syllabes.
