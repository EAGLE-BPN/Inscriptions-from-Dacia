Description

Object type: Autel votif ou base de statue (?)
Material: calcaire
Type of Inscription: Inscription votive?
Execution Technique: sculptées
Letters: 2.2
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: Dédiée probablement pour le salut de l'empereur Caracalla (212-217).

Bibliography:

  • N. Gudea, AMP 6, 1982, p. 71, no 6, fig. 6 (photo)
  • AE 1983, 857
  • idem, AMP 13, 1989, p. 766, no 27, p. 1135, pl. CCLXXII (photo)

Text

[- - - - - -
- - - Part(hici) max(imi)
Brit(annici) max(imi) ] Germ(anici)
[max(imi) pont(ificis)?ma]ximi
5[[. ] [ ] [ ]].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

…, (grand vainqueur des Parthes( ?), grand vainqueur des Bretons(?), grand) vainqueur des Germains), grand (pontife), ….


Commentary:

En raison des lettres de petites dimensions, il s’agit probablement d’une dédicace pour le salut d’un empereur. Celui-ci pourrait être Caracalla, avec le pontificat après les épithètes impériales.
