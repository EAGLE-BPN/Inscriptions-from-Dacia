Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: 4.5
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: À partir de Septime Sévère, en raison du statut municipal de Porolissum.

Bibliography:

  • I.Piso, R. Zăgreanu, D. Deac, ms.

Text

[[. ] [ ] [ ]]
inianus
d(ecurio) m(unicipii) P(orolissensis) IIII
viṛalis
5[[. ] [ ] [ ]].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

…inianus, décurion du municipium Porolissense, ancien quattuorvir, ….


Commentary:

Le dédicant portait un cognomen dérivé d’un nomen fini en –inius ou d’un cognomen fini en –inus, comme Licinianus ou Valentinianus ; voir H. Solin, O. Salomies, Repertorium, p. 459-460.
