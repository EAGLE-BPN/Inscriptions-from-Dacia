Description

Object type: Chaton d'anneau
Material: bronze
Type of Inscription: Gravure sur un chaton d'anneau
Execution Technique: gravées dans le métal
Letters: 0,2
Provenance: Potaissa
Findspot: Turda/Torda, Roumanie
Date: À partir de 168, quand la legio V Macedonica est venue en Dacie.

Bibliography:

  • M. Bărbulescu, Inscriptions, p. 262-263, no 67, fig. 150 a-b (photo, dessin).

Text

Ψυχ(ή)

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Psyche(?)


Commentary:

Petite plaque ovale en bronze, en fait le chaton d’un anneau, sur laquelle on a gravé un dessin représentant un lion tourné vers gauche, devant le lion deux signes difficilement à détérminer et la lettre Ψ, au-dessus une petite croix et au-dessous les lettres YX. Lecture de Bărbulescu. La même représentation apparaît souvent sur les pierres gravées.
