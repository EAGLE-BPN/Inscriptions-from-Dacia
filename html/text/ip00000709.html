Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tegula mammata
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 5,5
Provenance: ?
Findspot: Gilău, Roumanie
Date: La troupe est venue en Dacie au début du règne d'Hadrien (117-138). Les tegulae mammatae signifient la construction d’une installation de chauffage dans le prétoire du commandant ou dans les thermes, opération datable au début du stationnement de la troupe à Gilău, donc probablement sous Hadrien ou sous Antonin le Pieux.

Text

Al(ae) S(ilianae).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à l'ala Siliana.


Commentary:

Estampille sans cartouche, appliquée à l’aide d’un instrument à marquer au fer rouge les chevaux et le bétail; lettres imprimées. Pour l'ala Siliana en Dacie voir C. C. Petolescu, Auxilia Daciae, p. 76-78.
