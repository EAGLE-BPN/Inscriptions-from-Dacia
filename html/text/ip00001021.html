Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: anneau en argent
Material: argent
Type of Inscription: gravure sur un anneau
Execution Technique: gravées dans le métal
Letters: 0,2
Provenance: Potaissa
Findspot: Turda/Torda, Roumanie
Date: À partir de 168, quand la legio V Macedonica est venue en Dacie.

Bibliography:

  • M. Bărbulescu, Inscriptions, p. 248-251, no 55, fig. 134 (photo)
  • AE 2012, 1231.

Text

[V]alẹ[ri](
Ati(ane ?)
viva(s).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À ta santé, Valerius( ?) Atianus( ?).


Commentary:

Anneau en argent, ayant sur chaque côté d’un chaton approximativement carré deux orifices en forme de pelta. Au chaton avait été fixée une clef en fer, maintenant disparue. L’inscription a été gravée sur trois lignes dans le chaton. Bărbulescu : AL / ATI / VIVA(s). L. 1 : on peut penser aussi à [C]alp(urni).
