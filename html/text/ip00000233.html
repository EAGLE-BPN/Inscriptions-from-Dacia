Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle funéraire
Material: calcaire?
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Probablement du IIIe siècle, en raison du nomen Aurelius pas accompagné par praenomen.

Bibliography:

  • CIL III 920
  • I. I. Russu, AISC 4, 1941-1943(1944), p. 202, no 46.

Text

D(is) M(anibus)
Aur(elius) Epicatiu [ s]
vixit n(is) LII
Aur(elius) Viator VI
5vixit annis X [ .]
Aur(elius) Corbulo
vixit annis XXII
[ [. ] [ ] [ ]].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

Aux dieux Mânes. Aurelius Epicatius a vécu 2 ans, Aurelius Viator a vécu .. ans, Aurelius Corbulo a vécu 22 ans. ….


Commentary:

Russu : Epicatius serait une forme latinisée de Epicadus ; pour celle-ci voir A. Mayer, Illyrier, p. 139.
