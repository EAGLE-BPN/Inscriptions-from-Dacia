Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation; hederae
Provenance: Germisara
Findspot: Geoagiu, Roumanie
Date: De 156/157-159, quand M. Statius Priscus s'est trouvé à la tête de Dacie Supérieure.

Bibliography:

  • A. Rusu, E. Pescaru, La politique édilitaire dans les provinces de l'Empire romain (IIe-IVe siècles apr. J.-C.). Actes du Ier Colloque Roumano-Suisse, Deva 1991, Cluj 1993, p. 202
  • AE 1993, 1342
  • I. Piso, Fasti provinciae Daciae I, Bonn 1993, p. 67, no 16/6.

Text

Genio Imp(eratoris)
Antonini
Aug(usti) Pii p(atris) p(atriae)
M(arcus) Statius
5[P]riscus leg(atus)
[A]ug(usti) pr(o) pr(aetore)
v(otum) s(olvit) l(ibens) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Au génie de l'empereur Antonin le Pieux Auguste, père de la patrie. Marcus Statius Priscus, légat de l'empereur en qualité de pro praetore (gouverneur), s'est acquitté de (son) voeu de bon gré (et) à bon droit.


Commentary:

M. Statius Priscus a gouverné la Dacie Supérieure dans les années 156/157-159; voir I. Piso, Fasti provinciae Daciae I, p. 66-73.
