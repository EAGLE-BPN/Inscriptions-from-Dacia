Description

Object type: Stèle funéraire, fragment

Type of Inscription: Inscription funéraire

Date: Comme Aulusanus pourrait être de condition pérégrine, du IIe siècle.

Bibliography:

Text

[- - - - - -
vix(it) n(is) ... mil(itavit)]
an(nis) XV Aulusanu
s (h)eres bene merent
5i posuit
arbitrante Ingenui(?).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

... (a vécu ... ans, a servi) 15 ans. Aulusanus, en qualité d'héritier, érigea (ceci) à celui qui a bien mérité, sous l'arbitrage d'Ingenuus.


Commentary:

Est conservée la partie inférieure du champ épigraphique. Le nom du défunt est perdu. Eres a été écrit sans H. L'héritier (Aulusanus) porte un nom thrace. Dans le testament on a prévu aussi un arbiter, qui a accompli les conditions du testament.
