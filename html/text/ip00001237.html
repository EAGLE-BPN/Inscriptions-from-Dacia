Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation 3-4
Provenance: Germisara
Findspot: Geoagiu, Roumanie
Date: Probablement du IIIe siècle, en raison de l'écriture et du nomen Aelius, pas accompagné par praenomen.

Bibliography:

  • O. Floca, MatArh 1, 1953, p. 757-758, fig 3
  • S. Sanie, Cultele orientale I, p. 269, no 72
  • IDR III/3, 221.

Text

Soli inv
icto sac
rum Ael(ius) Iulius (votum) s(olvit)
l(ibens) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Consacré au Soleil invaincu. Aelius Iulius s'est acquitté (de son voeu) de bon gré (et) à bon droit.


Commentary:

Par Sol invictus il ne faut pas comprendre Mithras, mais bien un Baal syrien (S. Sanie).
