Source: UNIVERSITATEA BABES BOLYAI


Author: Rada Varga

Description

Object type: plaque
Material: marble
Type of Inscription: votive inscription
Execution Technique: carved
Letters: no
Provenance: Histria
Findspot: in a late phase of the curtain wall (k), Romania
Date: 3rd C BC

Bibliography:

  • ISM I 112

Text

[Καλλικ]ράτης Καλλικράτου καὶ οἱ στρατιῶ[ται π]επλευκότες ἐπὶ βοήθεια[ν Ἀ]πολλωνιαταῖς Διοσκόροις σωτῆρσι

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Callicrates, the son of Callicrates and the soldiers who sailed with him to help the people of Apollonia, dedicated the monument to the Dioscuri Saviors
