Description

Object type: Brique
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson

Provenance: ?
Findspot: Cluj, Roumanie
Date: À partir du règne d’Hadrien (118/138), en raison du statut de ville de Napoca.

Bibliography:

  • P. Iambor, Șt. Matei, A. Halasu, AMN 18, 1981, p. 144-145, n. 11, p. 148, pl. VII

Text

R(ei) p(ublicae) N(apocensis).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la communauté de Napoca


Commentary:

La brique a été produite sur un domaine appartenant à la ville de Napoca, affermée à un conductor; pour les cas connus de Sarmizegetusa voir I. Piso, EphEp 6, 1996, p. 162-164, 196-197 = Nordgrenze, p. 304-306, 341.
