Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: À partir du début du règne de Septime Sévère, en raison du rang municipal de la ville.

Bibliography:

  • C. Daicoviciu, AISC I / 2, 1928-1932, p. 61, no 1 b
  • idem, Dacica, Cluj 1970, p. 21, no 1 b
  • AE 1934, ad 17
  • R. Ardevan, Viața municipală în Dacia romană, Timișoara 1998, p. 428, no 437
  • ILD 466.

Text

[ I(ovi)] o(ptimo) m(aximo)
[ co]nservato(ri)
Ạel(ius) Super{i} Aug(ustalis)
muni(cipii) Sep(timii) {M}
5Pot(aissensis) v(oto) l(ibens) p(osuit).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

(À Jupiter) très bon(et) très grand, le sauveur. Aelius Super, Augustale du municipe Septimium Potaissa érigea(ceci) de bon gré à la suite d’un vœu.
