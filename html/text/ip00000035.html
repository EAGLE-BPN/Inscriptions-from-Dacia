Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: grès
Type of Inscription: Inscription votive
Execution Technique: primitivement sculptées
Letters:
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: En raison de l'écriture primitive et du manque du praenomen, probablement du IIIe siècle.

Bibliography:

  • O. Hirschfeld, SAWW 77, 1874, p. 379, no 9
  • EphEp II, 383
  • CIL III 7682.

Text

Voto
Libero pa
tri An(n)ius
Saturni
5nus.


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À la suite d’un vœu, à Liber le père. Annius Saturninus.
