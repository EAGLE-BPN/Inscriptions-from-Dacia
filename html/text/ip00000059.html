Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: grès?
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Peut-être du milieu du IIIe siècle ap. J.-C., en raison du statut colonial de la ville.

Bibliography:

  • I. Téglás, AÉrt 3, 191, p. 4, fig. (dessin), p. 48
  • R. Ardevan, Viața municipală, p. 427, no 434
  • ILD 08.

Text

[ [. ] [ ] [ ]]
IIvir col(oniae)
Pot(aissensis) v(otum) l(ibens) s(olvit).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

…, duumvir de la colonie de Potaissa, s’est acquitté de(son) vœu de bon gré.


Commentary:

Le problème ici c’est le rang colonial de Potaissa. Ulpien affirme qu’il aurait été octroyé par Septime Sévère(Dig. 0, 1, 1, 8-9). Pourtant, le rang colonial est attesté dans cette seule inscription, tandis que pour le rang municipal, accordé avec certitude par Septime Sévère, on a au moins six attestations. Il serait possible que le rang colonial ait été octroyé vers le milieu du IIIe siècle
