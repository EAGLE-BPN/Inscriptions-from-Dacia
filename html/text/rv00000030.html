Description

Object type: column
Material: marble
Type of Inscription: honorific inscription
Execution Technique: carving
Letters: 2,5
Provenance: Ulpia Traiana Sarmizegetusa
Findspot: forum vetus, est of the basilica, Romania

Bibliography:

  • C. Daicoviciu, 1927-1932, p. 518, 543, no1

Text

Ordini
Augustalium
P(ublius) Antonius
Super dec(urio) col(oniae)
5Sarm(izegetusae) metro-
polis
d(ono) d(edit).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

To the ordre of the Augustales. Publius Antonius Super, decurion of the colonia Sarmizegetusa, metropolis; he gave it.
