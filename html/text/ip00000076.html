Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: conglomérat calcareux
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: À partir de l'arrivée de la légion en Dacie, vers 168.

Bibliography:

  • C. Torma, AEM 3, 1879, p. 0-1, no 32
  • CIL III 7672.

Text

Iul(ius) Maxi
mianus(centurio)
genio(centuriae)
v(otum) s(olvit) l(ibens) m(erito).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Iulius Maximianus, centurion, s’est acquitté de(son) vœu de bon gré(et) à bon droit au génie(protecteur) de la centurie.


Commentary:

Voir pour ce centurion M. Bărbulescu, Legiunea V Macedonica, p. 70, no 1.
