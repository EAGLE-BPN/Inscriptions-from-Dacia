Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tuile fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire et graffites
Execution Technique: imprimées avant cuisson et incisées avant cuisson
Letters: 1,85,58,5
Provenance: Potaissa
Findspot: Turda/Torda, Roumanie
Date: À partir de 168, quand la legio V Macedonica est venue à Potaissa

Bibliography:

  • M. Bărbulescu, Inscriptions, p. 226-227, no 36, fig. 100 (dessin)
  • AE 2012, 1224.

Text

A R N B
L(egionis) V M(acedonicae)
L(egionis) V M(acedonicae).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

ARNB. Appartenant à la legio V Macedonica. Appartenant à la legio V Macedonica.


Commentary:

Graffites avant cuisson sur une tuile fragmentaire ; estampille légionnaire ; un graffite de quatre lettres au-dessus de l’estampille, un second graffite au-dessous d’elle, avec le même nom de légion. Exercice d'écriture.
