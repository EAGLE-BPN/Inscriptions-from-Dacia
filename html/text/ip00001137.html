Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Brique fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 4,8
Provenance: Tibiscum
Findspot: Jupa, Roumanie
Date: À partir de la fin du règne d'Antonin le Pieux, quand les numeri Palmyrenorum furent constitués.

Bibliography:

  • M. Moga, AMN 7, 1970, p. 136-138, fig. 1
  • M. Moga, I. I. Russu, Lapidarul Muzeului Banatului. Monumente epigrafice romane, Timișoara 1974, no 57
  • IDR III/1, 256.

Text

N(umeri) P(almyrenorum) T(ibiscensium).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant au numerus Palmyrenorum Tibiscensium.


Commentary:

Ligature NPT. Les numeri Palmyrenorum furent constitués vers la fin du règne d'Antonin le Pieux; voir pour cette troupe D. Benea, P. Bona, Tibiscum, București 1994, p. 54-56; C. C. Petolescu, Auxilia Daciae, București 2002, p. 139-140.
