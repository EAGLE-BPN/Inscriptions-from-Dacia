Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: hederae
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: Du IIIème siècle, en raison du statut municipal de Porolissum.

Bibliography:

  • D. Dana, R. Zăgreanu, Tyche 28, 2013, p. 32-35, no 2, pl. 4 (photo).

Text

I(ovi) o(ptimo) m(aximo)
et Iunoni
reg(inae) P(ublius) Ael(ius)
Stadylo
55 dec(urio) aedil(is)
pro voto.


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon (et) très grand et à Junon la reine. Publius Aelius Stadylo, ecurion (et) édile (érigea ceci) conformément au vœu.


Commentary:

Le nom Stadylo était inconnu jusqu’à présent. Il s’agit très probablement du grec Σταδύλων (plutôt que Στάδυλος), pas encore connu, construit sur la racine σταδ- (σταδαῖος : se tenant debout) ; voir la discussion chez D. Dana, R. Zăgreanu, p. 34-35.
