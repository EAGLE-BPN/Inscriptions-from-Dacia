Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: tuf volcanique
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: À partir du début du règne d’Hadrien, quand l’aile est venue en Dacie.

Text

Fortunae
reduci Ael(ius)
Celer praef(ectus)
eq(uitum) alae Fro
5nt(onianae) ob res
titutio
nem bali
nei posuit.


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À la Fortune qui fait revenir(qui te ramène sain). Aelius Celer, préfet des cavaliers de l’ala Frontoniana, érigea(ceci) à l’occasion de la restauration des bains.


Commentary:

L. 4- : division incorrecte en syllabes ; l. 7-8 : balineum, forme parallèle à balneum. On remarque la fonction religieuse que les thermes pouvaient parfois accomplir. Le même officier apparaît dans AE 2006, 1128 avec le prénom T(itus) ; voir encore H. Devijver, PME I, A 28.
