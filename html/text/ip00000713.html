Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tuile fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 4,5
Provenance: Samum
Findspot: Cășei, Roumanie
Date: La cohors I Britannica (milliaria) se trouve en Dacie à partir de 106.

Bibliography:

  • J. Szilágyi, Ziegelstempel, p. 55, pl. XVII/245-246
  • N. Gudea, LimesDac, p. *52, no 28/1
  • D. Isac, Samum 1997, p. 41, pl. II/7, 10
  • idem, Viața cotidiană, p. 22, 28, pl. IV/1, 4.

Text

C(ohortis) I B(ritannicae).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la cohors I Britannica.


Commentary:

Estampille sans cartouche, appliquée à l’aide d’un instrument à marquer au fer rouge les chevaux et le bétail; lettres imprimées. Voir pour la cohors I Britannica (milliaria) C. C. Petolescu, Auxilia Daciae, p. 86-87.
