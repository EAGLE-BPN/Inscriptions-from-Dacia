Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: hedera
Provenance: Optatiana
Findspot: Sutor, Roumanie
Date: Du IIIème siècle, en raison du nom érasé d'un empereur.

Bibliography:

  • I. I. Russu, AMN 5, 1968, p. 459-460, no 6, fig. 6 (photo, dessin)
  • AE 1971, 393
  • ILD 756.

Text

I(ovi) o(ptimo) m(aximo)
[pr]o sal(ute) d(omini) n(ostri)
[[- - -
- - -]] Aug(usti)
5[[. ] [ ] [ ]].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

À Jupiter très bon (et) très grand, pour le salut de notre seigneur … Auguste. ….


Commentary:

Dans les l. 3-4 a été martelé le nom d’un seul empereur du IIIème siècle, qui pourrait être Élagabal, Sévère Alexandre, Decius ou Aemilianus. Le dédicant aura été un commandant de la troupe qui stationnait à Optatiana.
