Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: anneau en argent
Material: argent
Type of Inscription: gravure sur un anneau
Execution Technique: gravées dans le métal
Letters: hederae 0,3
Provenance: Potaissa
Findspot: Turda/Torda, Roumanie
Date: Du IIIe siècle, en raison de la formule utere felix.

Bibliography:

  • M. Bărbulescu, Inscriptions, p. 250-251, no 57, fig. 138-140 (photos)
  • AE 2012, 1233.

Text

Utere felix.

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Sers t’en heureux.


Commentary:

Anneau en argent présentant douze cannelures extérieures, avec les lettres gravées dans les espaces entre les cannelures. Un espace entre E et L est vide.
