Description

Object type: Relief votif

Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: 0.50.8
Provenance: -
Findspot: Pojejena, Roumanie
Date: Du IIème siècle, en raison du nom de facture pérégrine.

Bibliography:

  • IDR III/1, 12.

Text

[[. ] []] Dizo Posidon(ii).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

..., Dizo, fils de Posidonius.


Commentary:

Du relief est conservé le coin inférieur droit; on y distingue deux scènes de la vie de Mithras. Le nom Dizo est thrace (D. Detschew, Die thrakischen Sprachreste2, Wien 1976, p. 135); le patronyme Posidonius (Ποσειδώνιος) est grec.
