Text

[D(is)] M(anibus)
Antonio Ero
ti patri vixi(t) an(nis)
LXXXXVIII posuer(unt)
55 Antonia Iulia
et Antonius E
ros fil(ii) b(ene) m(erenti) p(osuerunt).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

(Aux dieux) Mânes. A Antonius Eros, (leur) père. Il a vécu 98 ans. (Lui) érigèrent (ceci) Antonia Iulia et Antonius Eros, (ses) enfants. Ils érigèrent (ceci) à celui qui a bien mérité.


Commentary:

Pour le nom grec Eros (῎Ερως) voir L. Ruscu, AMN 35, 1998, p. 163 ; il trahit peut-être une condition d’affranchi. L. 5 : le nomen Iulia est utilisé ici comme cognomen. L. 5, 7 : le verbe posuerunt est utilisé deux fois.
