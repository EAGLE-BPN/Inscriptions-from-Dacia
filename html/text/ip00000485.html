Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance:
Findspot: Ciugud, Roumanie
Date: Du IIème siècle, en raison du nom de facture pérégrine du dédicant.

Bibliography:

  • L. Ruscu, CIGD 24
  • I. Piso, AMN 39-40, 2002-2003, p. 213-215, no 13, fig. 3 a-b (photo, dessin)
  • AE 2003, 1476.

Text

Κλυ(- - -)
Λόνγο
ς Ἀσια
[νοῦ ...].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

zoilianus


Commentary:

CIGD 24: Κλ(α)ύ(διος) / Λόνγος. Κλυ(- - -) pourrait être le nom d'une divinité inconnue d'Asie Mineure. Une rivière de Phrygie (Pline, NH 5, 108) portait le nom de Cludrus (= Κλύδρος).
