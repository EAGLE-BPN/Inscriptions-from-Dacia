Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Probablement du IIIe siècle, en raison du manque du praenomen et de l’écriture.

Bibliography:

  • CIL III 891.

Text

I(ovi) o(ptimo) m(aximo)
Val(erius) Rom(anus?)
Aur(elius) Fel(i)x
v(otum) s(olverunt) l(ibentes) m(erito).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon(et) très grand. Valerius Romanus(?)(et) Aurelius Felix se sont acquittés de(leur) vœu de bon gré(et) à bon droit.
