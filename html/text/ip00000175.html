Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: tuile
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: en relief
Letters: 0.91.2
Provenance: Sarmizegetusa
Findspot: Sarmizegetusa, Roumanie
Date: Il n'y a pas des critères de datation.

Bibliography:

  • I. Piso, Ephemeris Napocensis 6, 1996, p. 16, no 16 = idem, Nordgrenze, p. 307, no 16
  • AE 1996, 1290

Text

Ἀφ(- - -) Φουσ(- - -)

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

Appartenant à Af... Fus...


Commentary:

Le nomen est Afinius, Afranius ou un nom voisin; le cognomen est Fuscus ou un dérivé.
