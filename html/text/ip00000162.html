Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle funéraire
Material: grès
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: Points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Du IIIe siècle, en raison du nomen Aurelius pas accompagné par praenomen.

Bibliography:

  • I. I. Russu, Activitatea Muzeelor, Cluj 19, p. 4-, no 4, fig. 4(dessin)
  • ILD 14.

Text

[ Dis M(anibus
Aur(elius) Proc]
ulu [ s v(ix(it) an(nis)]
VII Auṛ(elius) [ Pro]
5culuṣ [ fil(io)]
pientis [ simo]
po [ s(uit)].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

(Aux dieux Mânes. Aurelius) Proculus(a vécu) sept(ans). Aurelius Proculus érigea(le monument)(au fils) très pieux.


Commentary:

L’enfant porte le cognomen de son père.
