Object type: Tuile fragmentaire

Provenance: Certiae

Date: Après l'arrivée, sous le règne d'Hadrien (117-138), de la cohors Ii Britannica en Dacie.

Bibliography:

Text

Coh(ortis) II Brit(annicae).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la cohors II Britannica.


Commentary:

Estampille en cartouche rectangulaire; les lettres en relief; typique est le signe après le T, qui peut être interprété soit comme une interpunction, soit comme un A.
