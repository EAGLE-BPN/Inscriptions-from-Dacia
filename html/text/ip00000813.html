Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tuile fragmentaire
Material: céramique
Type of Inscription: Graffite
Execution Technique: creusées
Letters:
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: Probablement du temps de Caracalla (212-217), quand le temple du dieu palmyrénien Bel a été reconstruit.

Bibliography:

  • N. Gudea, Porolissum 1986, p. 59, fig. XIV/1
  • idem, AMP 13, 1989, p. 174-175
  • idem, LimesDac, p. *49, no 25/31-33
  • I. Piso, D. Deac, ms.

Text

N(umeri) [P(almyrenorum)].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant au numerus (Palmyrenorum).


Commentary:

Pour la reconstruction du temple de Bel, le suprême dieu palmyrénien, voir AE 1980, 755.
