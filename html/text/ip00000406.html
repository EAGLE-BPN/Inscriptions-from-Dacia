Text

Χα[ῖρε]
Ἡδύλ[ος]
Θασ[ίου]
[[. ] [ ] []].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Salut à toi! Hedulos, (fils de) Thasios, ....


Commentary:

W. Pape, Wörterbuch, p. 453 : Ἡδύλος ou ῞Ηδυλος ; signifie « petit trésor ». Dans la l. 3 la solution de I. I. Russu, Θασ[ίος], a été suivi par tous les auteurs ; L. Ruscu (AMN 35, 1998, p. 166) : « Hedylos aus Thasos » ; eadem (CIGD) remarque que l’inscription de Porolissum contient le seul anthroponyme grec de Dacie avec l’indication de l’origine. Pourtant, Θασίος peut très bien être nom propre (CIL VI 21303: P. Licinius C(aiae) l. Thasius; voir W. Pape, Wörterbuch, p. 482 ; H. Solin, Personennamen I, p. 625).
