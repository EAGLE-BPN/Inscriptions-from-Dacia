Text

Mercu
rio vo
tum sol
vit Ma
5essia
nus.


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Mercure s’est acquitté de(son) vœu Maessianus.


Commentary:

Mommsen, d’après Weszprém : Mercu / rio vo / tum so [ l / v]it M / essia / nus. Russu corrigea la lecture d’après Pap Szatmáry, en admettant la forme Maessianus à la place du correcte Messianus, dérivé de Messius.
