Text

Terrae matrị
c(ives) R(omani) per
[ S]atrianuṃ
[ et] Ḍẹcumum
5[ m]ag(istros).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À la Terra mater. Les citoyens Romains(érigèrent le monument) par Satrianus(et) Decumus, leur magistri.


Commentary:

Les mêmes cives Romani, représentés par les mêmes magistri Satrianus et Decumus, érigèrent aussi AE 1992, 1468 = 1993, 1332.
