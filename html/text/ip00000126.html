Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle funéraire
Material: tuf volcanique?
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: Probablement du IIIème siècle, en raison du nomen Aurelius pas accompagné par praenomen.

Bibliography:

  • K. Torma, ErdMúz 3-4, 1866, p. 4, no 18
  • CIL III 802
  • S. Nemeti, Classica et Christiana / 2, 20, p. 414, no 31
  • C. Gaiu, R. Zăgreanu, Inscripții și piese sculpturale din castrul roman de la Ilișua, Cluj-Napoca 2011, p. 8, no I.3.b., p. 126, pl. VIII.

Text

[ [. ] []]
Aur(elius) Dec [ [. ] []]
vix(it) [ ann(is) ..]
me [ ns(ibus) ..]
5Ma [ [. ] []]
vix(it) an [ n(is) ..]
Aur(elius) De [ [. ] []] dec(urio) alae [ et [. ] []]
tina m [ ater fil(iis) pientis]
simis po [ suerunt].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

… Aurelius Dec… a vécu …(ans), … mois. Ma… a vécu … ans. Aurelius De…, décurion d’aile(avec) …tina(leur) mère erigèrent(ceci)(aux enfants) très pieux.


Commentary:

L. 1, C. Gaiu, Zăgreanu : Aur(elius) dec(urio), solution impossible, car il s’agit d’un cognomen comme Decimus, Decoratus etc. ; l. 3, Torma, Mommsen : ME ; l. 7-9, Mommsen : dec(urio) alae [ Front(onianae) - - -] / tina m [ ater filiis pientis] / simis po [ suit].
