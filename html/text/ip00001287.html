Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: andésite
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: Points de séparation; hederae
Provenance: Micia
Findspot: Vețel, Roumanie
Date: Comme la cohorte ne porte pas d'épithète impériale, du IIe siècle.

Bibliography:

  • C. Torma, AEM 3, 1879, p. 108, no 59
  • CIL III 7848
  • IDR III/3, 78.

Text

I(ovi) O(ptimo) M(aximo)
coh(ors) II F[l(avia)]
Com(magenorum) cu[i]
praeest
5C(aius) Pompo
nius Cas
sianus praefect(us)
v(otum) s(olvit) l(ibens) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon (et) très grand. Cohors II Flavia Commagenorum, à la tête de laquelle se trouve Caius Pomponius Cassianus en tant que préfet, s'est acquitté de (son) voeu de bon gré (et) à bon droit.


Commentary:

C. Pomponius Cassianus était chevalier romain et, en tant que préfet, exerçait la première milice équestre. La cohors II Flavia Commagenorum stationnait dans le camp de Micia; voir C. C. Petolescu, Auxilia Daciae, Bucuresti 2002, p. 97-99.
