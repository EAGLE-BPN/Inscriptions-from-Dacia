Author: Florian Matei-Popescu

Description

Object type: Stele
Material: Limestone
Type of Inscription: Funerary
Letters: 5-8
Provenance: Drobeta
Findspot: ?, Romania
Date: First half of the third century - the nomen Aurelius

Bibliography:

Text

D(is) M(anibus)
Aur(elius) Gratus
vixit annis
LXXX. Aur(elius) Gratus
5filius patri
b(ene) m(erenti) posuit.
Bonas vi-
as viator.
