Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle funéraire,
Material: tuf volcanique?
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: Probablement du IIIème siècle, en raison du nomen Aurelius pas accompagné par praenomen.

Bibliography:

  • K. Torma, ErdMúz 3-4, 1866, p. 47, no 22
  • CIL III 812
  • S. Nemeti, Classica et Christiana / 2, 20, p. 416-417, no 41
  • C. Gaiu, R. Zăgreanu, Inscripții și piese sculpturale din castrul roman de la Ilișua, Cluj-Napoca 2011, p. 84, no I.3.b.6, p. 12, pl. VII.

Text

[ [. ] [ ] [ ]]
[ Aur(elius) [. ] []]ius vet(eranus)
[ vixit anni]s LX Aur(elius)
[ [. ] []]anus fil(ius)
5[ patri b(ene)] m(erenti) [ p(osuit) ?].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

…(Aurelius) …ius, vétéran,(a vécu 60 ans. Aurelius …anus,(son) fils,(érigea ceci)(au père qui a bien) mérité.


Commentary:

Le défunt avait probablement servi dans l’ala Frontoniana.
