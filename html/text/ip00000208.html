Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Plaque honorifique encastrée dans une base
Material: marbre
Type of Inscription: Inscription honorifique
Execution Technique: sculptées
Letters: Points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: La pièce est datable du règne de Septime Sévère(193-211) ou, plus probablement, du règne de Caracalla(212-217).

Bibliography:

  • M. Bărbulescu, Inscriptions, p. 94-99, no 9, fig. 29-38.

Text

[ Imp(eratori) Caes(ari)] Ḍịṿ [ i] Ḷ(ucii) Sept(imii) S [ everi A]ug(usti) Aṛ [ abici? Adiabenici Parthici max(imi)? fil(io) M(arco) Aur(elio) Antonino p(io) fel(ici) Aug(usto)?]
[ [. ] [ ] [ ]]
PR [ - - - leg(io) V Mac(edonica)?]
suḅ [ - - - - - -

- - -]GO [ - - - - - -

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

(À l’empereur César Marcus Aurelius Antoninus, le pieux, l’heureux, Auguste, fils de?) Lucius Septimius Severus Auguste, vainqueur des Arabes(?),(vainqueur des Adiabènes, grand vainqueur des Parthes?), ...(la Ve légion Macedonica?), sous l’autorité de ..., préfet et ..., a(ont) érigé(ceci).


Commentary:

Bărbulescu suppose, peut-être à juste titre, que le monument ait été dédié à Septime Sévère. Pourtant, la filiation(Divi) suggère qu’il puisse s’agir plutôt de Caracalla. Frg. f : deux personnages ou un seul personnage avec deux fonctions; frg. j : le prédicat posu [ it] ou posu [ erunt] est peu fréquent dans ce type de dédicaces(Bărbulescu).
