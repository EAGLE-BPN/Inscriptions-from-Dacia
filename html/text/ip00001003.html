Indici
Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tuile fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire et graffite
Execution Technique: imprimées avant cuisson et incisées avant cuisson
Letters: 2,52,2
Provenance: Potaissa
Findspot: Turda/Torda, Roumanie
Date: À partir de 168, quand la legio V Macedonica est venue en Dacie.

Bibliography:

  • M. Bărbulescu, Inscriptions, p. 224-225, no 32, fig. 96 (photo)
  • AE 2012, 1220.

Text

L(egionis) V M(acedonicae).Victorinus.

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la legio V Macedonica.Victorinus


Commentary:

Graffite avant cuisson sur une tuile fragmentaire, sur laquelle on avait appliqué une estampille légionnaire. L’estampille, rétrograde, est en cartouche rectangulaire, avec une bordure intérieure en relief ; les lettres en relief ; manque l’extrémité gauche.
