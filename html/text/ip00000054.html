Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Il n'y a pas assez de critères pour dater l'inscription, car Iulianus peut être nom personnel et, tout aussi bien, cognomen.

Bibliography:

  • I. I. Russu, Activitatea Muzeelor 19, p. 3-4, no 3, fig. 3(dessin)
  • AE 196, 268
  • ILD 491.

Text

Silvano
domes
[ ti]co Iu
lianus
5v(otum) s(olvit).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Silvain domesticus(gardien de la maison). Iulianus s’est acquitté de(son) voeu.


Commentary:

Le dédicant semble appartenir à la couche inférieure.
