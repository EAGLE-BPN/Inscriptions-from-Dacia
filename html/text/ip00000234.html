Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle ou base de monument funéraire,
Material: grès
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Probablement de la fin du IIe siècle ou du IIIe siècle, en raison du nomen Aurelius.

Bibliography:

  • CIL III 912.

Text

[ [. ] [ ] [ ]]][)
[ [. ] []]AII(?)
[ [. ] []] Aur(elius) [ …]
[ Secun?]dus mag(ister)
5[ Aug(ustalis) Nap]ocae Aur(eliae)
Felicissimae so
rori b(ene) m(erenti) p(osuit).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

…, Aurelius … Secundus(?), magister Augustalis(président de l’ordre des Augustales) de Napoca, érigea(ceci) à Aurelia Felicissima,(sa) sœur qui a bien mérité.


Commentary:

On a suivi la lecture de Mommsen. Pour les magistri Augustales voir CIL III 862, pour le nom Felicissima voir H. Solin, O. Salomies, Repertorium, p. 330. Le texte présente encore des problèmes : le nom devrait se trouver au début et pas dans les l. 4- ; le nombre supposé de lettres ne permet pas d’ajouter dans la l. 4 le rang de la colonie, ce qui est inhabituel.
