Description

Object type: Stèle funéraire, fragment

Type of Inscription: Inscription funéraire

Date: Probablement du IIe siécle, en raison du nom de facture pérégrine du défunt.

Bibliography:

Text

D(is) [Manibus)]
Appianu[s ... S]
urus vix[it]
an[nis] XXXX An[t(onius) Ru]
5sticus [...]
adfinis fec(it).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Aux dieux (Mânes). Appianus Surus a vécu 40 ans. Antonius Rusticus ..., parent, a fait (ceci).


Commentary:

Appianus est le nom personnel du défunt, tandis que Surus, le patronyme, devrait se trouver au génitif. Dans la l. 4 la ligature AN se trouve sur le bord de la stèle. Les lettres sont peintes en rouge.
