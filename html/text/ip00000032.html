Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: La pièce est probablement contemporaine des inscriptions AE 1992, 1468 = 1993, 1332 et AE 1992, 1469 = 1993, 1332 et est, donc, datable des années 168-193 / 211. Le nom du centurion, contenant tous les éléments, indique tout aussi bien le IIe siècle.

Text

I(ovi) o(ptimo) m(aximo)
et dis hospitiḅ(us)
huiusce loci
M(arcus) Pollius M(arci) f(ilius)
5Ouf(entina) Hispanus
Medi [ ola]no
(centurio) ḷ [ eg(ionis) V] Ṃ(acedonicae) p(iae) c(onstantis)
et Caed [ i]cia Marci
a eius cum suis.


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon(et) très grand et aux dieux hospitaliers de ce sanctuaire. Marcus Pollius Hispanus, fils de Marcus, inscrit dans la tribu Oufentina, originaire de Mediolanum, centurion de la(Ve) légion Macedonica, la pieuse(et) la constante, et Caedicia Marcia, sa(femme)(érigèrent le monument) avec les siens.


Commentary:

Locus signifie un sanctuaire qui ne respectait pas toutes les conditions d’un temple ; une interprétation différente chez les premiers auteurs. La legio V Macedonica reçut les épithètes p(ia) c(onstans) sous Commode ; voir E. Ritterling, RE XII / 2(192), 180. Pour le centurion M. Pollius Hispanus voir encore M. Bărbulescu, Legiunea V Macedonica, p. 68-69, no 4. Pour le rare nomen Desticia voir W. Schulze, Eigennamen, p. 293 ; H. Solin, O. Salomies, Repertorium, p. 68.
