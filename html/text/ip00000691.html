Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Brique de suspensura
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 2,52,7
Provenance: Praetorium
Findspot: Mehadia, Roumanie
Date: En Dacie la cohors III Delmatarum est attestée pour la première fois par le diplôme militaire RMD II 123 = AE 1987, 843 le 1 avril 179. Elle avait la garnison à Praetorium (Mehadia).

Bibliography:

  • J. Szilágyi, Ziegelstempel, p. 52, pl. XV/209
  • N. Gudea, Tibiscus 4, 1975, p. 122, fig. 32/1
  • IDR III/1, 100
  • M. Macrea, N. Gudea, I. Moțu, Praetorium, p. 50, 161, pl. IX/25-26
  • N. Gudea, LimesDac, p. *31, no 15/2.

Text

Coh(ortis) III D(elmatarum).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la cohors III Delmatarum.


Commentary:

Estampille sans cartouche, avec les lettres imprimées. Pour la cohors III Delmatarum voir C. C. Petolescu, Auxilia Daciae, p. 102-103.
