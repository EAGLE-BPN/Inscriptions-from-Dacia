Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Plaque honorifique encastrée dans une base
Material: marbre
Type of Inscription: Inscription honorifique
Execution Technique: sculptées
Letters: Points de séparation
Provenance: Sarmizegetusa
Findspot: Sarmizegetusa, Roumanie
Date: C. Arrius Antoninus a gouverné la Dacie en 175(?)-177.

Bibliography:

  • AE 1931, 122
  • IDR III/2, 86
  • I. Piso, Fasti provinciae Daciae I. Die senatorischen Amtsträger, Bonn 1993, p. 107, no 4.

Text

C(aio) Arrio
Antonino
fil(io) C(ai) Arri(i) Anto-
nini leg(ati) Augus(ti)
5pr(o) pr(aetore)
col(onia) Ulp(ia) Traian(a)
Aug(usta) Dac(ica) Sarm(izegetusa).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Caius Arrius Antoninus, fils de Caius Arrius Antoninus, légat de l'empereur en qualité de pro praetore (gouverneur). La colonia Ulpia Traiana Augusta Dacica Sarmizegetusa.


Commentary:

Voir pour cet important sénateur de l'époque des Antonins I. Piso, Fasti I, p. 106-117.
