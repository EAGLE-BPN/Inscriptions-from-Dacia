A P C R H

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Commentary:

Graffite et dessin sur un fragment de tuile (19 x 16 x 2,7 cm) cassé en deux. Dans la partie supérieure ont été gravées avant cuisson quelques lettres. Au-dessous a été gravé, toujours avant cuisson, un personnage dont les traces sont à peine esquissées, avec les cheveux suggérés par de brèves lignes verticales. Des lignes semblables suggèrent un col. Les lettres ne donnent aucun sens.
