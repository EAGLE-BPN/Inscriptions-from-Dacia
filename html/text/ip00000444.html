Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Samum
Findspot: Cășei, Roumanie
Date: Le consulat de Ap. Claudius Iulianus pour la seconde fois et de Bruttius Crispinus a été exercé en 224 (A. Degrassi, Fasti consolari, p. 62).

Bibliography:

  • I. I. Russu, ActMuz 2, 1956, p. 125-127, no 3, fig. 3 (dessin)
  • AE 1957, 328
  • E. Schallmayer et alii, Osterburken I, p. 407, no 528
  • D. Isac, Osterburken II, p. 212, no 7, p. 213, pl. I/7.

Text

I(ovi) o(ptimo) m(aximo) Dọ[l(icheno)]
Scantius
Lucius b(ene)f(iciarius) co(n)s(ularis)
agens curam
5stationis ex
vot(o) [posui]t.

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

À Jupiter très bon (et) très grand, Dolichenus. Scantius Lucius, bénéficiaire du consulaire, detenant la responsabilité du poste de garde, érigea (ceci) à la suite d’un vœu.


Commentary:

Pour la statio beneficiariorum voir C. Daicoviciu, AMN 3, 1966, p. 165-166 ; D. Isac, dans : Osterburken II, p. 205-211 ; idem, Samum, p. 48-58. Dans le nom de l’officier Lucius, à l’origine praenomen ou nomen, joue le rôle de cognomen ; pour le nomen Scantius voir W. Schulze, Eigennamen, p. 143 ; H. Solin, O. Salomies, Repertorium, p. 164. L’officier apparaît encore dans AE 1957, 327 et dans ILD 778.
