Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Plaque base de monument funéraire,
Material: tuf volcanique?
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: Probablement à partir de la fin du IIème siècle, en raison des formules finales.

Bibliography:

  • K. Torma, ErdMúz 3-4, 1866, p. 44, no 16
  • CIL III 80
  • S. Nemeti, Classica et Christiana / 2, 20, p. 414-41, no 34
  • C. Gaiu, R. Zăgreanu, Inscripții și piese sculpturale din castrul roman de la Ilișua, Cluj-Napoca 2011, p. 80-81, no I.3.a., p. 124, pl. VI.

Text

D(is) M(anibus)
[ [. ] []] Bonosa
[ vix(it) ann(is)] XLII Val(erius) Vale
[ rianus] vet(eranus) ex de(curione) al(ae) Fro(ntonianae)
5[ coniu]gi pientissimae
[ b(ene)] m(erenti) f(aciendum) c(uravit).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Aux dieux Mânes. … Bonosa(a vécu) 42(ans). Valerius Valerianus, vétéran, ancien décurion de l’ala Frontoniana, prit le soin de faire(ceci) à celle qui a(bien) mérité.


Commentary:

Le cognomen Bonosa est répandu dans les provinces danubiennes ; voir OPEL I, 311. Le même Valerius Valerianus apparaît dans CIL III 808.
