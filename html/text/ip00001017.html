Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Anneau en fer
Material: fer
Type of Inscription: gravure sur un anneau
Execution Technique: gravées dans le métal
Letters: 0,20,3
Provenance: Potaissa
Findspot: Turda/Torda, Roumanie
Date: Du IIIe siècle, en raison de la formule utere (felix).

Bibliography:

  • M. Bărbulescu, Inscriptions, p. 248-249, no 54, fig. 133 (photo)
  • AE 2012, 1230.

Text

Ut(ere) Val(eri).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Sers t’en (heureux), Valère.


Commentary:

Anneau en fer (selon M. Bărbulescu en argent), peut-être d’abord argenté, avec une inscription incisée sur le chaton oval. Bărbulescu : Ut(ere) f(elix), Val(eri), mais le F n’existe pas, Le sens est néanmoins celui de utere felix.
