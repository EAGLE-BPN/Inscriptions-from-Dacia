Description

Object type: Brique d'hypocauste fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: point de séparation 1,31,8
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: Oon se trouve aux débuts de la province, sous Trajan (Szilágyi) ou sous Hadrien (V. Moga, Apulum 10, 1972, p. 159-160); voir aussi Gudea. On a probablement eu besoin de briques afin de construire les principia ou le prétoire de Pomet.

Text

Leg(ionis) XIII Ge(minae).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la legio XIII Gemina.


Commentary:

Estampille en cartouche rectangulaire, avec le nom de la légion flanquée par des anses en relief; les lettres en relief; deux barres, l’une au-dessus, l’autre au-dessous le numéral. Le type est identique à IDR III/6, 95. Ressemble au type suivant, où la barre se trouve pourtant seulement au-dessous du numéral. À notre avis les briques de la legio XIII Gemina ont été apportées d’Apulum dans une situation dans laquelle leur fabrication semblait être plus difficile que le transport.
