Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: En principe, à partir de l'arrivée en Dacie de la Ve légion Macedonica

Bibliography:

Text


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Au génie(protecteur) de l’arsénal érigea(ceci) Aelius Vitalis, gardien des(armes).


Commentary:

L. 2-3, Bărbulescu : arma / menta(rii). L’inscription est importante parce qu’elle indique l’endroit exact de l’armamentarium ; voir M. Bărbulescu, Legiunea V Macedonica, p. 144-14 ; c’est le custos armorum qui en était responsable.
