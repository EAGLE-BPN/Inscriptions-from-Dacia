Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle funéraire,
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters:
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: À partir d’Antonin le Pieux, quand ont été constitués les numeri Palmyrenorum.

Bibliography:

  • C. Daicoviciu, Cahier de fouilles, p. 3 (dessin).

Text

[[. ] [ ] [ ]]
[[. ] []]BASAV
[[. ] []] n(umeri) P(almyrenorum) P(orolissensium)
[[. ] []] coni
5[ugi pien]tissi
5 [mae b(ene) ?] ṃ(erenti)p(osuit).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

… …, du numerus Palmyrenorum Porolissensium, … érigea (ceci) à l’épouse qui a (bien) mérité.


Commentary:

L. 1 : nom palmyrénien impossible à identifier. Il s’agit d’un militaire du numerus Palmyrenorum Porolissensium, qui érigea un monument funéraire à son épouse.
