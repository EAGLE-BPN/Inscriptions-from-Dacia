Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tuile fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: hederae, point de séparation 2,42,7
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: Probablement des années 213-217, quand la vexillation de la legio VII Gemina a remplacé des troupes de Porolissum, qui ont accompagné Caracalla dans la guerre parthique.

Text

L(egionis) VII G(eminae) f(elicis).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la legio VII Gemina felix.


Commentary:

Selon I. Piso, une vexillation de la legio VII Gemina de l'Espagne Tarraconnaise aurait remplacé à Porolissum des troupes qui ont accompagné Caracalla dans la guerre parthique. Estampille imprimée avec un signaculum aproximativement rectangulaire, visible lorsqu’on le pressait assez, comme dans notre cas; lettres imprimées.
