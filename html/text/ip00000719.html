Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Brique d'hypocauste
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 1-1
Provenance: ?
Findspot: Bologa, Roumanie
Date: À partir du règne d’Hadrien (117-138), quand la cohors I Aelia Gaesatorum est arrivée en Dacie.

Bibliography:

  • N. Gudea, Bologa 1997, p. 26, 84, fig. 16 (deux dessins)
  • idem, LimesDac, p. *40, no 21/5.

Text

C(o)ho(rtis) I Ae(liae) G(aesatorum).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la cohors I Aelia Gaesatorum


Commentary:

Estampille avec les lettres imprimées. Le signaculum, qui peut à la rigueur être interprété comme cartouche, était de forme rectangulaire; il peut être distingué dans la mesure où on le pressait; dans le cas de cette estampille on en distingue toutes les marges. Voir pour la cohors I Aelia Gaesatorum (milliaria) C. C. Petolescu, Auxilia Daciae, p. 103-104.
