Indici
Source: UNIVERSITATEA BABES BOLYAI


Author: Florian Matei-Popescu

Description

Object type: Altar
Material: Limestone
Type of Inscription: Dedication
Letters: yes
Provenance: Carsium
Findspot: Piua Pietrii, Romania
Date: Second half of the second century - fist half of the third century

Bibliography:

  • Gr. Tocilescu, AEM 8, 1884, p. 34, no. 4
  • CIL III 7493
  • ISM V 127

Text

Genio cent(uriae)
Fl(avii) Ianuari(i)
Fl(avius) Avitianus
sig(nifer) n(umeri) Suro-
5[r]um s(agittariorum) eius
voto libye(ns) (sic!)
posuit

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

To the Genius of Flavius Ianuarius' centuria, set by Flavius Avitianus, standard bearer of his numerus of the Syrian archers, willingly fulfilling the vow.
