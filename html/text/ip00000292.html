Description

Object type: Autel votif ou base de statue
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: 4.5
Provenance: Potaissa
Findspot: Turda/Torda, Roumanie
Date: À partir de 168, date de l’arrivée de la légion en Dacie.

Bibliography:

  • I. Téglás, AÉrt 22, 1902, p. 82
  • R. Münsterberg, J. Oehler, JÖAI 5, 1902, p. 105-106, no 5 (dessin).

Text

[[. ] [ ] [ ]]
[[. ] []]n(ius) [.]roc[…]
[si]g(nifer) leg(ionis) V Mac(edonicae) [p(iae)]
[cu]m Firmini[o et]
5[P]risca filiis e[t]
Vera uxor(e).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

…, signifer de la legio V Macedonica la pieuse, avec Firminius et Prisca, ses enfants, et Vera, son épouse.


Commentary:

Firminius, qui joue ici le rôle d’un cognomen, est dérivé du nom Firminus est suggère plutôt une origine d’une province celtique occidentale.
