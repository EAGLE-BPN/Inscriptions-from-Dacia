Description

Object type: Tuile fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 2,3
Provenance: ?
Findspot: Gilău, Roumanie
Date: À partir du début du règne d'Hadrien (117/118), quand l'ala Siliana est venue en Dacie.

Bibliography:

  • I. Szilágyi, Ziegelstempel, p. 51/XIV/199, pl. XIV/199

Text

Al(ae) S(ilianae).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à l'ala Siliana.


Commentary:

Estampille en forme de tabula ansata, avec les lettres en relief. Szilágyi regarda la pièce comme étant fragmentaire; au contraire, Isac et Gudea la considérèrent complète, en la lisant Al(ae) S(ilianae). À Gilău on a découvert un exemplaire complet.
