Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle funéraire,
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: À partir de 168, quand la legio V Macedonica arriva en Dacie.

Bibliography:

  • I. Téglás, AÉrt 28, 1908, p. 360
  • ILD 18.

Text

[ [. ] [ ] [ ]]
[ [. ] []] vix(it)
[ annis … Dom]itius Re
[ - - - mil(es) leg(ionis) ?] s(upra) s(criptae) filius EO
5[ [. ] [] vix(it) anni]s XI Domi
[ tius [. ] []]us filius
[ [. ] []] p(osuit).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

…, a vécu …(ans) ; Domitius Re…,(soldat de la légion) ci-dessus mentionnée(?),(son) fils ; …(a vécu) 11 ans. Domitius …us, en tant que fils, … érigea(ceci).


Commentary:

On a trois défunts : un premier Domitius, probablement légionnaire, un Domitius Re [ - - -], le fils du premier, probablement militaire dans la même légion, et un enfant âgé de 11 ans. Le monument fut érigé par un quatrième Domitius, en tant que fils et frère.
