Source: UNIVERSITATEA BABES BOLYAI


Author: Florian Matei-Popescu

Description

Object type: Altar
Material: Limestone
Type of Inscription: Votive
Letters: yes
Provenance: Histria
Findspot: Reused in the Late Roman fortification, Romania
Date: 166 - 172 (169) AD - based on the Marcus Aurelius' imperial title

Bibliography:

  • Al. Suceveanu, RevRoumHist 14, 1975, p. 111-118
  • ISM I 358

Text

[...pro sal(ute)] Imp(eratoris) [M(arci) Aur(eli)]
Antonini
Aug(usti) Armen-
ia(ci) Part(h)i(ci) Medi(ci), vic(us)
5Arcidava per A[...]

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

For the good health of the Emperor Marcus Aurelius Antoninus Augustus, the defeater of the Armenians, Parthians and Medians, the village Arcidava (set) through the care of A[- - -].
