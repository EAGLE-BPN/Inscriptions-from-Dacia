Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation 3-3
Provenance: Germisara
Findspot: Geoagiu, Roumanie
Date: Probablement des règnes d'Antonin le Pieux, de Marc Aurèle ou de Commode, quand ont été érigées les autres inscriptions des thermes de Germisara.

Bibliography:

  • I. Piso, A. Rusu, Revista monumentelor istorice 59/1, 1990, p. 16, no 10
  • AE 1992, 1486.

Text

Nymphis
sacr(um)
L(ucius) Livius Mar
cellus IIvi
5ral(is) col(oniae) pro
fili(is) ex vi
so l(ibens) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Consacré aux nymphes. Lucius Livius Marcellus, ancien duumvir de la colonie, (érigea ceci) pour ses enfants à la suite d'un rêve; (il le fit) de bon gré (et) à bon droit.


Commentary:

L. Livius Marcellus était duumvir de la colonie de Sarmizegetusa. Germisara appartenait au territoire de cette colonie.
