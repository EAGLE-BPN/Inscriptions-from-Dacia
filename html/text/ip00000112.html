Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Base de statue impériale,
Material: tuf volcanique
Type of Inscription: Inscription honorifique
Execution Technique: sculptées; l. 6 martelée.
Letters: points de séparation
Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: a. 244-249, règne de Philippus Arabs

Bibliography:

  • V. Wollmann, R. Ardevan, dans : Fontes historiae. Studia in honorem Demetrii Protase(éd. C. Gaiu, C. Găzdac), Bistrița – Cluj-Napoca 2006, p. 670-671, no 4, p. 677, fig. 7-8(photo et dessin)
  • AE 2006, 1127

  • I. Piso, Revista Bistriței 27, 2013, p. 49-50, no 13, fig. 13 a-b (photo, dessin).

Text

[ [ [ Imp(eratori) Caes(ari)]
[ M(arco) Iulio Phi]
[ li]ppo [ Aug(usto)]]]
ala Fr [ onto]
5nian [ a [ [ Phi]]]
[ [ lippia [ na]] de]
vota [ numi]
ṇi mạ [ iesta]
[ tique eius].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

(À l’empereur César Marcus Iulius Philippus Auguste), l’ala Frontoniana Philippiana dévouée à son numen(componente divine de la personne impériale)(et a sa) majesté.


Commentary:

La largeur d’environ 80 de la base justifie la supposition qu’il s’agit d’une base de statue pédestre.
