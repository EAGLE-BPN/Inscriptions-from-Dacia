Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tuile
Material: céramique
Type of Inscription: estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 1.6
Provenance: Potaissa
Findspot: Turda/Torda, Roumanie
Date: De 222-235, en raison de l’épithète Severiana.

Bibliography:

  • M. Grec, IstDacPor, p. 136, 150, fig. 83 (dessin).

Text

[L(egionis) V] M(acedonicae) p(iae) f(idelis) S(everianae).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

Appartenant (à la legio V) la pieuse (et) la fidèle, Severiana.


Commentary:

Estampille en cartouche avec les marges latérales arrondies. M. Grec y voit une ligature MA.
