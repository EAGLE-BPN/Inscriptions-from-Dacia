Description

Object type: Stèle ou plaque funéraire,
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: Points de séparation
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: À partir de Septime Sévère, en raison du statut municipal de Porolissum.

Bibliography:

Text

[- - - - - -
v]ịx(it) [ann(is) …]
Ael(ius) [- - -
v]et(eranus) IA[- - -
5m]un(icipii) [- - -
- - - - - -]


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

…, a vécu (… ans). Aelius …, … du municipe ….


Commentary:

Il s’agit d’un vétéran qui est parvenu à une dignité municipale.
