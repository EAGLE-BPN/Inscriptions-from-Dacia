Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: l. : hederae 6.8
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Par analogie avec AE 1992, 1469 = 1993, 1332; AE 1992, 1470 = 1993, 1332, trouvées dans le même sanctuaire, probablement de l'intervalle 168-193 / 211.

Bibliography:

  • M. Bărbulescu, A. Cătinaș, EN 2, 1992, p. 113, no 2, fig. 7(photo, dessin)
  • iidem, dans : Politique édilitaire dans les provinces de l'Empire Romain - IIème-IVème siècles après J. -C.(Colloque roumano-suisse Deva 21-26 octobre 1991) Cluj-Napoca 1993, p. 0, no 1, p. 62, fig. 7(photo, dessin)
  • AE 1992, 1467
  • 1993, 1332 .

Text

Merc(urio)
Aug(usto) s [ ac(rum)?]
D(ecimus) Iun(ius)
Teles
5v(otum) s(olvit) l(ibens) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

À Mercure Auguste. Decimus Iunius Teles s’est acquitté de(son) vœu de bon gré(et) à bon droit.


Commentary:

On remarque le cognomen Teles d’origine grecque(Τέλης).
