Text

Μηνὶ ἀνει
{ι}κήτῳ Aur(elius)
Marcianuṣ
ṣignifer
5lẹg(ionis) voto

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

À l’invaincu Men. Aurelius Marcianus, signifer de la légion, érigea(ceci) à la suite d’un vœu.


Commentary:

C’est C. Daicoviciu(RE XXII / 1(193), 19) qui a reconnu dans le dessin de Téglás cette divinité lunaire micro-asiatique : MENI ANEIKHTΩ ; cf. Gostar : Μηνὶ ἀνει / {ι}κή [ τ]ῳ ; J. et L. Robert : Μηνὶ ἀνει / {ι}κή(τ)ῳ; Russu: Μηνὶ ἀνε / ικήτω; pour Men voir A. Lesky, RE XV / 1(1931), 689-697 ; pour le culte de cette divinité en Dacie voir Al. Popa, Apulum 6, 1967, p. 14-161. L’épithète ἀνίκητος(invictus) n’est pas connu pour Men, ce qui indique une possible contamination avec Mithra ou Helios(Daicoviciu, Gostar). Russu n’exclue pas une erreur du lapicide. L., Gostar, J. et L. Robert, Lane : l [ e]g(ionis) ; Petolescu: leg(ionis) ; Bujor, AE : l(egionis) G(eminae) ; pourtant, entre le L et le G semble se trouver un H, donc un E grec.
