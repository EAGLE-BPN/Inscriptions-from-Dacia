Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle funéraire,
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: Points de séparation 3.5
Provenance: Tibiscum
Findspot: Jupa, Roumanie
Date: À partir de la fin du règne d'Antonin le Pieux, quand le numerus Palmyrenorum Tibiscensium fu constitué.

Bibliography:

  • M. Moga, Lapidarul Muzeului Banatului, Timișoara 174, no 26
  • IDR III/1, 155.

Text

[D(is)] M(anibus)
[Ae]l(ius) Male
[[. ] []]E mil(es) e
[x(?) n(umero) Pa]l(myrenorum) vix(it) an(nis)
5[[. ] []]ZI[..]A
[[. ] [ ] [ ]].

Translation:

(Aux dieux) Mânes. Aelius(?) Male, ..., miles (du numerus) Palmyrenorum, a vécu ... ans. ....


Commentary:

Le nom Male est palmyrénien (I. I. Russu).
