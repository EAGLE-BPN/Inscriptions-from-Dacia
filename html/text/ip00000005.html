Description

Object type: Autel votif
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: 161-180 ou, plutôt, 211-217.

Bibliography:

  • CIL III 877
  • M. Bărbulescu, AMN 14, 1977, p. 177, no 3.

Text

Herculi
invicto
pro sal(ute) M(arci)
[ A]ur(elii) Ant [ o]
5[ nini Aug(usti)]
[ [. ] [ ] [ ]].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À l’invaincu Hercule pour la santé de(l’empereur) Marcus Aurelius Antoninus(Auguste) ….


Commentary:

La lecture d’après Mommsen. Du nom de l’empereur manquent Imp. Caes. Il s’agit de Marc Aurèle ou de Caracalla, plutôt du dernier.
