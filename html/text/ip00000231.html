Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle ou plaque funéraire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance:
Findspot: Moldovenești / Várfalva, Roumanie
Date: À partir de l’arrivée, vers 168, de la legio V Macedonica en Dacie.

Bibliography:

  • CIL III 93.

Text

Aur(elius) C [ api?]
to tesse(rarius)
leg(ionis) V M(acedonicae)
vix(it) n(is)
5XL Ael(ia)
[ Cand]ida
[ - - - - - - ?].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

Aurelius Capito(?), tesserarius de la legio V Macedonica, a vécu 40 ans. Aelia Candida … ?


Commentary:

Le tesserarius est le soldat qui transmet le mot d’ordre du commandant ; pour le tesserarius legionis voir A. v. Domaszewski, Rangordnung2, p. 43-44.
