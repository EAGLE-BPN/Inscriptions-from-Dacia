Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tuile fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 1,7
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: Les numeri Palmyrenorum furent constitués vers la fin du règne d’Antonin le Pieux (C. C. Petolescu, SCIVA 26/3, 1975, p. 426-427).

Bibliography:

  • ILD 744.

Text

N(umeri) P(almyrenorum) P(orolissensium).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant au numerus Palmyrenorum Porolissensium.


Commentary:

Estampille en cartouche légèrement trapézoïdale; les lettres en relief. On pourrait envisager aussi la lecture: A(lae) n(umeri) P(almyrenorum) P(orolissensium); cf. ἔπαρχος εἴλης ἱππέων ἀριθμοῦ Παλμυρηνῶν Πορολυσσηνῶν (ILS 9472 = IG XII/II 1, 146) et Al(ae) n(umeri) Il(l)yr(icorum) (CIL III 6284 = 8074, 7 = ILD 454). Pourtant, on ne peut distinguer aucune ligature AL.
