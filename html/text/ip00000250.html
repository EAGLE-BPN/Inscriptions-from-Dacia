Description

Object type: Stèle funéraire,
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Probablement encore du IIe siècle, en raison de la titulature(Mihăilescu-Bîrliba) et de la belle écriture.

Text

Timocle [ s Caes(aris) n(ostri)]
ab instrumeṇ [ tis]
vix(it) an(nis) XXXV Bona
Caes(aris) n(ostri) ser(va) nepoti
5posuị [ t].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Timocles, archiviste(de notre empereur), a vécu 3 ans. Bona, esclave de notre empereur, érigea(ceci) à son neveu(petit fils).


Commentary:

Timocles, portant un nom grec(Τιμοκλῆς, voir W. Pape II, p. 129), surveillait les actes du tabularium du procurateur.
