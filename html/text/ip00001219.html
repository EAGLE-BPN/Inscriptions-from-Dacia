Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: Points de séparation 6-8
Provenance: Germisara
Findspot: Geoagiu, Roumanie
Date: En raison des autres dédicaces de Germisara, l'empereur pourrait être Antonin le Pieux (138-161).

Bibliography:

  • CIL III 1394
  • ILS 7152
  • IDR III/3, 234.

Text

Herculi
invicto
pr(o salute) inpera
toris col(legium)
5Galatarum
L(ucius) Livius Ma
rcellinus d(ono) d(edit) d(edicavitque).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Hercule invaincu, pour (le salut) de l'empereur, le collège des Galatae. Lucius Livius Marcellinus en a fait on (et) a dédié.


Commentary:

Les Galatae proviennent de l'Asie Mineure. L. 3-4: inpera/toris à la place de impera/toris.
