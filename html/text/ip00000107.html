Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Base de statue
Material: tuf volcanique
Type of Inscription: Inscription honorifique
Letters: points de séparation
Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: De 131, en raison du chiffre de la puissance tribunicienne ; l’épithète Antoniniana remonte à Caracalla et date probablement de 213.

Text

[ Imp(eratori) Cae]s(ari) divi Traian [ i]
[ Parthi]ci filio divi Nẹṛ [ vae]
nepoti
Traiano Hadriano A [ ug(usto)]
5pont(ifici) maximo trib(unicia) pot(estate) XV
co(n)s(uli) III p(atri) p(atriae)
ala I Tungrorum Fronton(iana)
Antoniniana.


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

(À l’empereur) César Trajan Hadrien Auguste, fils du divin Trajan Parthique, petit-fils du Divin Nerva, grand pontife, investi avec la puissance tribunicienne pour la XVème fois, consul trois fois, père de la patrie, l’ala I Tungrorum Frontoniana Antoniniana(érigea ceci).


Commentary:

Le texte de Protase contient quelques petites erreurs de lecture qui se sont perpétuées chez tous les autres auteurs : [ Imp(eratori) Caesar]i divi Traian(i) / [ Parthic]i filio divi N [ ervae] / nepoti / Traiano Hadriano [ Aug(usto)] / pont(ifici) maximo trib(unicia) pot(estate) X [ V ?] / co(n)s(uli) III p(atri) p(atriae) / ala I Tungrorum Frontoni(ana) / Antoniniana. Le résultat le plus important de la réexamination de la pièce est que le chiffre XV de la puissance tribunicienne est certain. L’épithète Antoniniana a été ajouté sous Caracalla(Protase), plus précisément en 213, à l’occasion de la visite de l’empereur, quand fut érigé aussi CIL III 79. Pour l’emplacement de la statue dans la cour des principia voir F. Marcu, The Internal Planning of Roman Forts of Dacia, Cluj-Napoca 2009, p. 80-81.
