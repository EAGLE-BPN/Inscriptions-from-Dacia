Text

[Imp(eratori) Caes(ari) M(arco)] Anto
[nio Gordia]no
[pio fel(ici) i]ṇvic
[to Aug(usto) pont]if(ici)
55 [max(imo) trib(unicia) p]o[t(estate)]
[[. ] [ ] [ ]].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

(À l’empereur César Marc) Antoine Gordien, (pieux, heureux et) invincible (Auguste, grand) pontife, investi avec la puissance (tribunicienne) ….


Commentary:

Le texte complet, inédit, chez C. Daicoviciu ; M. Macrea et les autres : [Imp(eratori) Caes(ari) M(arco)] Anto / [nio Gordia]no / [pio fel(ici) in]vic / [to Aug(usto) - - -].
