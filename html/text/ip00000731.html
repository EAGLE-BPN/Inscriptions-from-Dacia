Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Brique
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 5,54,5
Provenance: ?
Findspot: Orheiul Bistriței, Roumanie
Date: À partir du règne d’Hadrien (117-138) ou d’Antonin le Pieux (138-161), quand le camp aura été reconstruit en pierre.

Bibliography:

  • M. Macrea, D. Protase, Șt. Dănilă, SCIV 18/1, 1967, p. 119, fig. 7/2

  • N. Gudea, LimesDac, p. 56, no 31/1-3.

Text

C(ohortis) I H(ispanorum) (milliariae).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la cohors I Hispanorum (milliaria).


Commentary:

Estampille sans cartouche, avec les lettres imprimées. Il s'agit de la cohors I Flavia Hispanorum (milliaria); voir pour cette troupe C. C. Petolescu, Auxilia Daciae, p. 111-112.
