Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées, lignes de guidage
Letters:
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: À partir du début du règne de Septime Sevère, quand Potaissa a reçu le rang municipal.

Bibliography:

  • Z. Milea, M. Jude, SCIV 23 / 4, 1972, p. 667-670, fig. 1(photo, dessin)
  • AE 1974, 0
  • ILD 490.

Text

Silvano d
omes{s}tic [ o]
Aurel(ius) Via [ to]
r scriba
5munic(ipii) Se [ p(timii)]
Pot(aissensis).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

À Silvain domesticus. Aurelius Viator, scribe du municipe Septimium Potaissa.
