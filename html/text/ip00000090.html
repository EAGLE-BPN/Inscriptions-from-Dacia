Description

Object type: Relief votif ou autel
Material: tuf volcanique
Type of Inscription: Inscription votive
Execution Technique: sculptées

Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: Du IIe siècle, si le dédicant avait un nom de facture pérégrine.

Text

[ [. ] []]
Ζαμαηνίσθης
εὐχὴν θεοῖς
βασι [ λ]ικοῖς εὐ(χήν).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

…, de Zama(?), s’est acquitté de son vœu(fait) aux dieux royaux. Il s’est acquitté de son vœu.


Commentary:

Ardevan, Wollmann, Nemeti, Gaiu, Zăgreanu: Ζαμαννίσθης / εὐχὶν θεοῖς / χ [ θονί]οις [ - - -] ; l. 1, AE(Piso): Ζαμαηνίσθης, fini en –θης à la place de της, désigne probablement l’origine du dédicant, par exemple de Zama(Ζάμα), située en Cappadoce, sur la route de Tavium à Mazaka(Ptol., 6, 11 ; TP X / 1 ; voir L. Zgusta, Kleinasiatische Ortsnamen, Heidelberg 1984, p. 177, no. 320, p. 30, no 112 / 1, p. 31, no 113). Dans ce cas le nom du dédicant se trouve au-dessus du relief. Il est pourtant possible qu’il s’agisse d’un nom personnel, cf. Ζαμαμειζῶν, Ζαμάσπης, Ζαμάσφης(W. Pape I, p. 437). L. 2, HD : εὐχήν ; l. 3 : la deuxième lettre, A, et les dernières cinq lettres, IKOIC, sont claires et alors la lecture probable est βασι [ λ]ικοῖς et pas χ [ θονί]οις. Il s’agit probablement de divinités difficilement à identifier de l’Asie Mineure. Il semble que le lapicide ait oublié qu’il avait écrit εὐχὴν dans la l. 2 et en a répété à la fin de la l. 3 les premières deux lettres.
