Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tuile fragmentaire
Material: céramique
Type of Inscription: Graffite avant cuisson
Execution Technique: incisées avant cuisson
Letters: point de séparation 2,2
Provenance: Potaissa
Findspot: Turda/Torda, Roumanie
Date: À partir de 168, quand la legio V Macedonica est venue en Dacie.

Bibliography:

  • M. Bărbulescu, PotaissaStMon, p. 152, pl. VII/4
  • idem, Inscriptions, p. 228-229, no 37, fig. 101 (photo)
  • AE 2012, 1225.

Text

Ạ Ḅ C D E F G H I
R S P

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Commentary:

Graffite avant cuisson sur le coin inférieur droit d’une tuile ; trois fragments jointifs ; les lettres écrites sur deux lignes. Exercice d’ecriture. Les lettres RSP de la l. 2 ne semblent avoir un rapport avec la l. 1.
