File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à Valerius Cat....


Commentary:

Branga, Russu (IDR) et Petolescu identifièrent Val. Cat. avec un procurateur financier de Dacie Porolissensis (CIL III 857), car un procurateur n’aurait pas manqué de mentionner son titre. Il s’agit d’un propriétaire terrien dans le territoire de Napoca, qui portait le cognomen Cat(ulus), Cat(ullus), Cat(ullinus) etc.
