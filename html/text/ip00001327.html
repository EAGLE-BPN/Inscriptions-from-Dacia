Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: Points de séparation
Provenance: Micia
Findspot: Vețel, Roumanie
Date: Probablement du IIIe siécle, en raison de l'épithète deux aeternus de la divinité.

Bibliography:

  • R. Münsterberg, J. Oehler, JÖAI 5, 1902, Bbl. 131, no 3
  • AE 1903, 68
  • C. Daicoviciu, ACMIT 1930-1931 (1932), p. 37, no 4
  • IDR III/3, 133.

Text

Silva
no deo
aeterno
Quinti
5anus d
ec(urio) alae
v(otum) s(olvit) l(ibens) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Silvain, dieu éternel. Quintianus, décurion d'aile, s'est acqutté de (son) voeu de bon gré (et) à bon droit.


Commentary:

Quintianus était décurion de l'ala I Hispanorum Campagonum, quiavait sa garnison dans le camp de Micia; voir pour cette troupe C. C. Petolescu, Auxilia Daciae, Bucuresti 2002, p. 72-73.
