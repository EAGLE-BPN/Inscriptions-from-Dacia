Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tuile fragmentaire
Material: céramique
Type of Inscription: Graffite
Execution Technique: Gravée après cuisson
Letters: 3,8
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: À partir des règnes de Septime Sévère et Caracalla, quand la cohors III Campestris est arrivée à Porolissum.

Bibliography:

  • I. Piso, D. Deac, ms.

Text

C(o)h(ortis) III.

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la cohors III Campestris.


Commentary:

C’est comme si on avait oublié d’appliquer l’estampille dans la pâte molle. On a gravé le nom de la troupe après cuisson. Le texte n’a pas de suite. Il s’agit donc d’équipes de la cohors III Campestris.
