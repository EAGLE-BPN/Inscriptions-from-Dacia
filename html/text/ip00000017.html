Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Il n'y a pas des critères pour la datation.

Bibliography:

  • I. I. Russu, Activitatea Muzeelor 19, p. 1-2, fig. 1(dessin)
  • AE 196, 266
  • M. Bărbulescu, A. Cătinaș, EN 2, 1992, p. 111, fig. 2(photo, dessin)

  • AE 1993, 1332

Text

I(ovi) o(ptimo) m(aximo)
L(ucius) Carvil
ius Stati
us v(otum) s(olvit) l(ibens) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

À Jupiter très bon(et) très grand. Lucius Carvilius Statius s’est acquitté de(son) vœu de bon gré(et) à bon droit.


Commentary:

Pour le nomen Carvilius voir W. Schulze, Eigennamen, p. 139, 403, 44 ; H. Solin, O. Salomies, Repertorium, p. 48. Le nomen Statius joue ici le rôle d’un cognomen. L. 2-3 : on remarque la division incorrecte en syllabes.
