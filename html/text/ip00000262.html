Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description


Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters:
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: À partir du règne d’Hadrien, en raison du nomen Aelius.

Bibliography:

  • I. Téglás, AÉrt 24, 1904, p. 412
  • G. Téglás, Klio 11, 1911, p. 02 f.

Text

[ [. ] [ ] [ ]]
[ [. ] []]it Ael(ius, -a)
[ [. ] [ ] [ ]].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Commentary:

[ Fec]it, [ posu]it ou [ vov]it. La pièce a probablement un caractère funéraire, mais le caractère votif n’est pas exclu. Il n’est pas clair, non plus, si [ - - -]it est le prédicat de Ael(ius, -a).
