Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Relief votif
Material: marbre
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: Points de séparation 1.5
Provenance: Tibiscum
Findspot: Jupa, Roumanie
Date: À partir de la fin du règne d'Hadrien, quand a été constitué le numerus Palmyrenorum Tibiscensium.

Bibliography:

  • D. Isac, I. Stratan, Banatica 2, 1973, p. 121, no 5
  • IDR III/1, 147.

Text

[N(umerus) P]al(myrenorum) et n(umersu) M(aurorum)
[vot]um l(ibenter) po[suer(unt)].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

Le numerus Palmyrenorum et le numerus Maurorum se sont acquittés de (leur) voeu de bon gré.


Commentary:

Le numerus Palmyrenorum et le numerus Maurorum stationnaient dans le grand camp de Tibiscum.
