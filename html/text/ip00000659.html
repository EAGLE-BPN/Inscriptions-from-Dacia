Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Brique d'hypocauste
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 1,5
Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: En principe à partir du règne d’Hadrien, quand la l’unité est venue en Dacie; si l’installation de chauffage a été construite dans le prétoire du commandant ou dans les thermes de l’unité, plutôt sous Hadrien ou sous Antonin le Pieux.

Text

Alae Front(onianae).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à l'ala Frontoniana.


Commentary:

Estampille en cartouche rectangulaire, ayant dans son intérieur une seconde cartouche, suggérée par deux bords horizontaaux; les bords de la cartouche intérieure et les lettres en relief; les lettres A, F et N sont collées aux bords de la cartouche intérieure.
