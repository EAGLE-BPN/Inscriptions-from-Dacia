Translation:

..., (pour le salut de nos empereurs) Sévère (et) Antonin, (nos) deux Augustes.


Commentary:

Bărbulescu : [PRO SAL(ute) IMPP(eratorum)] / [L(ucii) SEPTIMI] / SEV(eri) [ET M(arci) AVR(eli)] / ANTO[NINI] / AVGG(ustorum). Selon cette reconstitution le texte aurait occupé presque toute la circonférence du fût, ce qui est impossible. D’ailleurs, la modalité de désigner les deux empereurs uniquement par les cognomina Severus et Antoninus est courante et en Dacie fréquemment utilisée (AE 1944, 74 = IDR III/3, 47 (Micia) ; CIL III 1343 = IDR III/3, 77 = AE 1978, 705 (Micia) ; CIL III 1564 = IDR III/1, 57 ; IDR III/1, 271 (Voislova); AE 1987, 848 = ILD 199 (Tibiscum) ; CIL III 14479 = IDR III/5, 426 (Apulum) ; CIL III 1127 = IDR III/5, 427 (Apulum) ; CIL III 1439 = IDR III/2, 328 (Sarmizegetusa)). Pour la l. 4 voir CIL VI 29844, 23 : [Cli]vus Victoria[e] Severi et Antonini Aug[g(ustorum)] nn(ostrorum). Le premier éditeur suppose qu’il s’y agisse d’un monument honorifique ou d’un édifice dans les principia, en remarquant aussi que cette colonne est beaucoup plus épaisse que les colonnes du portique. Dans ce cas les choses sont claires : il s’agit d’une base en forme de colonne probablement pour Jupiter, sinon pour la Victoire. Le dédicant n’est pas nommé. Dans ce cas la dédicace aura été faite par le primus pilus, qui veillait sur l’édifice du commandement.
