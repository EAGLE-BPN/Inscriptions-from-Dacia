Description

Object type: Autel ou base de statue funéraire
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: Points de séparation 4.5
Provenance: Tibiscum
Findspot: Jupa, Roumanie
Date: À partir de la fin du règne d'Antonin le Pieux, quand le numerus Palmyrenorum Tibiscensium fut constitué.
