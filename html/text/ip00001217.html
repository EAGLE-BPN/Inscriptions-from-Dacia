Text

I(ovi) o(ptimo) m(aximo)
C(aius) Roman
ius Summus
de(curio) col(oniae) v(otum) s(olvit) m(erito).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon (et) très grand. Caius Romanius Summus, décurion de la colonie, s'est acquitté de (son) voeu à bon droit.


Commentary:

Uroiu se trouvait sur le territoire de la colonie de Sarmizegetusa. Le cognomen Summus est fréquent dans les provinces celtiques occidentales.
