Description

Object type: Statue base
Material: Limestone
Type of Inscription: Dedication
Letters: No
Provenance: Troesmis
Findspot: The Western Late Roman fort, Romania
Date: 165-170 AD, when Pontius Laelianus was governor of Lower Moesia

Bibliography:

  • CIL III 774
  • 6182

Text

M(arco) Pontio
[L]aeliano
[c(larissimo)] v(iro) patri Pont(ii)
Laeliani
5[l]eg(ati) Aug(usti) pr(o) pr(aetore)
[or]do Troesm(ensium)


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

In the honour of Marcus Pontius Laelianus, highly regarded man, the father of Pontius Laelianus, imperial legate. The Council of Troesmis.
