Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Stèle funéraire
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: hedera
Provenance:
Findspot: Mihai Viteazul / Alsó-Szentmihályfalva, Roumanie
Date: Il n'y a pas des critères de datation.

Bibliography:

  • C. Torma, AEM 3, 1879, p. 3, no 42
  • CIL III 7701.

Text

[ D(is)] M(anibus)
[ [. ] [] Val]eria vixit
[ an(nis) [. ] []] Viator
[ [. ] [] coniu]gi posuit
5[ [. ] [ ] [ ]]

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

(Aux dieux) Mânes. … Valeria a vécu …. … Viator érigea(ceci) à … épouse …. ?


Commentary:

L. 2 : Valeria semble jouer le rôle de cognomen ; l. 4 : commençait probablement par une épithète.
