Text

T(ito) Flavio Longin[o]
Q(uinto) Marcio Turboni
leg(ato) Aug(usti) pr(o) pr(aetore)
Titius Crispus
5cornicular(ius)
eius

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

(In the honor of) Titus Flavius Longinus Quintus Marcius Turbo, the governor (of Lower Moesia), set by Titius Crispus, his chief of staff.
