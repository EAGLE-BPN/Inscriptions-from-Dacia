Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: marbre
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: Points de séparation
Provenance: Tibiscum
Findspot: Jupa, Roumanie
Date: Lecanius Paulinus apparaît encore dans IDR III/1, 137, datée de 211-212, où il porte le nome supplémentaire Aurelius de Caracalla. Cela veut dire que la dédicace à Jupiter a été faite avant 211/212.

Bibliography:

  • M. Moga, I. I. Russu, Lapidarul Muzeului Banatului, Timișoara 1974, no 13: IDR III/1, 138.

Text

I(ovi) o(ptimo) m(aximo)
Lecanius
Paulinus
vet(eranus) v(otum) s(olvit).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon (et) très grand. Lecanius Paulinus, vétéran, s'est acquitté de son voeu.


Commentary:

Dans IDR III/1, 137 le nomen apparaît dans la forme Laecanius.
