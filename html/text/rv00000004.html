Source: UNIVERSITATEA BABES BOLYAI


Author: Rada Varga

Description

Object type: plaque
Material: marble
Type of Inscription: building inscription
Execution Technique: carved
Letters: 9-1
Provenance: Ulpia Traiana Sarmizegetusa
Findspot: forum vetus, Romania
Date: 116-117, if indeed the emperor bears the epithets of optimus and Parthicus

Bibliography:

Text

[Imp(eratori) Caes(ari) Diui Neruae fil(io) Ne]r[uae] Tr[aiano]
[Opti?]mo [Au]g(usto) [Germ(anico) Dac(ico) Parth(ico)? pontif(ici)] m[ax(imo) trib(unicia)]
[potest(ate)---] P O[---] V[---]
[col(onia) Ulpia Traiana Aug(usta) D]ac(ica) Sa[rmizegetusa]
5[condit]o[ri s]uo.


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

To the emperor Caesar Nerva Traianus, son of the divine Nerva, optimus, Augustus, Germanicus, Dacicus, pontifex maximus, invested with the tribune's power, ... the colonia Ulpia Traiana Augusta Dacica Sarmizegetusa to its founder.
