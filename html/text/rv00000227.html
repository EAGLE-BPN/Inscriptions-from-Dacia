Source: UNIVERSITATEA BABES BOLYAI


Author: Rada Varga

Description

Object type: altar / statue base
Material: limestone
Type of Inscription: votive inscription
Execution Technique: carved
Letters: yes 5-6
Provenance:
Findspot: in secondary position on a stone barrack from the praetentura sinistra, Romania
Date: 135-138

Bibliography:

  • AE 1983, 859

Text

[Imperatori Caesari Divi]
[Traiani D]ạcị[ci Parthi-]
[ci fil(io) Divi Ne]rvae nep[oti Tra-]
[iano H]adriano
5[Aug(usto) pontif(ici)] max(imo) trib(unicia) pot(estate) X[…]
[imp(eratori) I]I co(n)s(uli) III
[ala Silian]a c(ivium) R(omanorum) torq(uata) et [armil(lata)]


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

To the emperor Caesar Traianus Hadrianus Augustus, son of the divine Trajan, conqueror of the Dacians, conqueror of the Parthians, pontifex maximus, holding the tribune's power for the....time, hailed as imperator 2 times, consul 3 times, the ala Siliana, formed of Roman citizens, decorated with a torques and a bracelet...
