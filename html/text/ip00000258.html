Author: Ioan Piso

Description

Object type: Stèle ou plaque funéraire,
Material: calcaire?
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: À partir de la fin du règne d’Antonin le Pieux, quand les numeri Maurorum ont été constitués.

Bibliography:

  • C. Torma, ArchKözl 3, 1863, p. 14, no 3
  • CIL III 911 = 769
  • O. Hirschfeld, SAWW 77, 1874, 382.

Text

[ [. ] [ ] [ ]]
[ Va ?]leria [ [. ] []]
o vix(it) an(nis) III m(ensibus) [ [. ] []]
T VAL dec(urio) n(umeri) M(aurorum) Ṃ(iciensium)
5natis piissimis [ [. ] []].

Translation:

… Valeria …, …o, a vécu 3 ans, … mois, … décurion du numerus Maurorum Miciensium(?)(érigea ceci) à(ses) très pieux enfants ….


Commentary:

L. 3, Aem. Hübner(apud Mommsen) : dec(urio) N(apocae) mu [ nicipii] ; Hirschfeld : dec(urio) n(umeri) m(ilitum) m…. ; A. v. Domaszewski pensa à juste titre à une troupe de Mauri. Pour le n(umerus) M(aurorum) M(iciensium) voir C. C. Petolescu, Auxilia, p. 136. Que cette troupe aurait jamais stationné à Potaissa(D. Tudor, Orașe, tîrguri și sate în Dacia romană. București 1968, p. 2) est une fausse hypothèse ; voir I. I. Russu, SCIV 23 / 1, 1972, p. 7. Le nom de l’officier ne peut pas être constitué de praenomen et de nomen ; on y a une lacune ou bien une erreur de transcription.
