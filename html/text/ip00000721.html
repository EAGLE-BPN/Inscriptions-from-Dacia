Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Tuile fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters: 1,72,7
Provenance: ?
Findspot: Bologa, Roumanie
Date: À partir du règne d’Hadrien (117-138) la cohors I Aelia Gaesatorum semble partager le camp de Bologa avec la cohors II Hispanorum.

Bibliography:

  • J. Szilágyi, Ziegelstempel, p. 11, p. 53/223, p. 64, pl. XV/223 (dessin)
  • N. Gudea, Bologa 1997, p. 84, fig. 16
  • idem, LimesDac, p. *40, no 21/5-6.

Text

C(o)h(ortis) I Ae(liae) G(aesatorum).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à la cohors I Gaesatorum.


Commentary:

Estampille avec cartouche en forme de tabula ansata, dextrograde, avec les lettres imprimées. On distingue une anse droite libre, tandis que l’anse gauche est suggerée par deux petits triangles. Il semble que les lettres du signaculum aient été très proéminentes, car les anses ne sont visibles que lorsqu’on applique profondement le signaculum, comme dans notre cas. Pour la cohors I Aelia Gaesatorum voir C. C. Petolescu, Auxilia Daciae, p. 103-104.
