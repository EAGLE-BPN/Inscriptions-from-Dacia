Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: Points de séparation
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: Du IIIème siècle, en raison de l’épithète deus.

Bibliography:

  • N. Gudea, E. Chirilă, AMP 16, 1992, p. 145, 169, fig. 12 (dessin)
  • ILD 686.

Text

[De]o Volkạno
[p]ro saḷutẹ n(umeri)
[Pal]ṃ(yrenorum) P[o]r(olissensium) [..]O
[[. ] []] X∙AM
5[[. ] [ ] [ ]].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

Au dieu Volcain, pour le salut du numerus Palmyrenorum Porolissensium( ?) ….


Commentary:

On a aussi assez d’indices pour reconnaître le numerus Palmyrenorum Porolissensium, ce qui est important pour la localisation de la troupe dans le camp de Pomet.
