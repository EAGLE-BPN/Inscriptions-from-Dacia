Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: andésite
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: Points de séparation
Provenance: Petrae?
Findspot: Bejan, Roumanie
Date: À partir du règne de Marc Aurèle, en raison de la séquence M. Aur(elius) dans le nom du dédicant.

Text

Hercul(i)
et Silva
no vex
illatio
55 l(egionis) XIII G(eminae) M(arcus)
Aur(elius) Ari
mo v(oto) m(erito) p(osuit)
immuni(s).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Hercule et à Silvain. La vexillation de la legio XIII Gemina (et) Marcus Aurelius Arimo, immunis, érigea (ceci) à bon droit.


Commentary:

Le cognomen Arimo pourrait être d'origine iranienne (Russu/IDR). La vexillation de la legio XIII Gemina, impliquée dans les travaux de Bejan, se trouvait sous l'autorité d'un immunis; voir pour cette charge A. v. Domaszewski, Rangordnung, p. 35, 37, 43.
