Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Ampelum
Findspot: Pătrânjeni, Roumanie
Date: Du IIIe siècle, en raison du nomen Aurelius pas accompagné par praenomen.

Bibliography:

Text

Fortunae
reduci et Ge
nio n(umeri) Aur(elius) Redux
praef(ectus) n(umeri) M(aurorum) H(ispanensium)
5d(onum) l(ibens) p(osuit).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À lal Fortune qui te ramène sain et au génie protecteur du numerus. Aurelius Redux, préfet du numerus Maurorum Hispanensium, a fait le don de bon gré.


Commentary:

Dans les anciennes publications, l. 5: n(umeri) M(aurorum) H(ispanorum, -ispalensium). H(ispanensium) signifierait que les soldats auraient été recrutés parmi les Mauri détachés en Espagne. Cette unité semble avoir stationné dans le camp très peu connu d'Abrud.
