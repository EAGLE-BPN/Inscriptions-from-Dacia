Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: marbre
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Sarmizegetusa
Findspot: Sarmizegetusa, Roumanie
Date: Probablement du IIIe siècle, en raison du nomen Aurelius pas accompagné par praenomen.

Bibliography:

  • G. Téglás, P. König, AEM 8, 1884, p. 54
  • CIL III 7910
  • IDR III/3, 236.

Text

I(ovi) O(ptimo) M(aximo)
Aurel(ius) Valens
Flavius Zora
dec(uriones) coll(egii) fabru(m)
55 ex vot(o) posue[r(unt)].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À Jupiter très bon (et) très grand. Aurelius Valens (et) Flavius Zora, décurions du collège des ouvriers, érigèrent (ceci) à la suite d'un voeu.


Commentary:

Le nom Zora est palmyrénien.
