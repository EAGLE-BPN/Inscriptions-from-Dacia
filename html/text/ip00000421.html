Description

Object type: Stèle funéraire,
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: 5.5
Provenance: Porolissum
Findspot: Moigrad, Roumanie
Date: S'il s’agit de la troupe, à partir de la fin du règne d’Antonin le Pieux ; s’il s’agit du municipe, à partir de Septime Sévère. La formule finale suggère plutôt le IIIème siècle.

Text

[- - - - - -
- - - P]or(olissensis)
[[. ] [] pie]nti
[ssimo ...].


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

... de Porolissum, ... au (à la) ... très pieux (pieuse).


Commentary:

L. 1, tous les auteurs : OR. La personne qui érigea le monument était soit militaire du numerus Palmyrenorum Porolissensium, soit dignitaire du municipium Septimium Porolissense.
