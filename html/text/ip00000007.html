Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Base de statue, le dé
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Par analogie avec CIL III 881, probablement à partir de l'empereur Commode.

Text

Isidi myrioni
mae C(aius) Iul(ius) Mar
tialis pater et
L. Livius Victori
5nus quaestor
collegio Isidis
d(ono) d(edit).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

À Isis celle aux dix mille noms. Caius Iulius Martialis, père, et Lucius Livius Victorinus, questeur, ont offert(ceci) comme don pour le collège d’Isis.


Commentary:

L. 1-2 : myrionimae à la place de myrionymae(μυριώνυμος, ον) ; voir pour cette épithète L. Bricault, dans : Hommage, p. 67-78 ; elle apparaît en Dacie aussi dans L. Vidman, SIRIS, no 692 = IDR III /, 4. Les deux dédicants remplissent des attributions dans le collège d’Isis ; pour le culte des divinités égyptiennes en Dacie voir S. Sanie, Arheoogia Moldovei 27, 2004, p. 61-82 ; idem, dans : Assaph -11, 200-2006(ΚΑΛΑΘΟΣ. Studies in Honor of Asher Ovadiah, éd. S. Mucznik), p. 1-124 ; M.-Ch. Budischovsky, dans : Isis en occident. Actes du IIème Colloque international sur les études isiaques, Lyon III, 16-17 mai 2002(éd. L. Bricault), Leiden – Boston 2004, p. 17-176 ; eadem, dans : Nile into Tiber, p. 26-288
