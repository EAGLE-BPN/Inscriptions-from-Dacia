Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: marbre
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: Points de séparation
Provenance: Tibiscum
Findspot: Jupa, Roumanie
Date: Du IIIème siècle, en raison de la carrière de P. Aelius Marius, flamen de la colonia Aurelia Apulensis.

Bibliography:

Text

S(oli) I(nvicto) N(abarze) M(ithrae)
pro salute
P(ublii) Ael(ii) Mari(i)
Hermadio
5act(or) Turran(ii)
Dii v(otum) s(olvit) l(ibens) m(erito).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

À l'invaincu Sol Nabarze Mithras pour le salut de Publius Aelius Marius. Hermadio, intendant de Turranius Dius, s'est acquitté de (son) voeu de bon gré et à bon droit.


Commentary:

L. 1: N(abarze) ou N(umini). Hermadio (Ἐρμαδίων) était l'intendant de Turranius Dius (Δῖος), qui apparaît aussi dans CIL III 1548 = IDR III/1, 141.
