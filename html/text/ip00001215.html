Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: calcaire
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: hederae 4,55,5
Provenance: Micia
Findspot: Vețel, Roumanie
Date: Vers 193, en raison de CIL III 1374 = IDR III/3, 45, datée de 193, dans laquelle apparaît le même préfet Scribonius Castus.

Bibliography:

Text

Fortunae
Aug(ustae) sacrum
Scribonius Castu[s praef(ectus)]
5 coh(ortis) I[I Fl(aviae) Com]
5m[ag(enorum)] - - -
- - -

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Consacré à la Fortune Auguste. Scribonius Castus, préfet de la cohors II Commagenorum ...


Commentary:

Dans CIL III 1374 = IDR III/3, 45 le préfet porte le nom complet de Sex. Boebius Scribonius Castus. Pour la cohors II Flavia Commagenorum, qui stationnait à Micia, voir C. C. Petolescu, Auxilia Daciae, Bucuresti 2002, p. 97-99.
