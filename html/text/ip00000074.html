Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Probablement du IIIe siècle, en raison du nomen Aurelius pas accompagné par praenomen.

Bibliography:

  • CIL III 893.

Text

[ I(ovi)] o(ptimo) m(aximo)
[ I]unoni
Aur(elius) Vic
[ t]or b(ene)f(iciarius) ex
5voto pos(uit).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

À(Jupiter) très bon(et) très grand(et) à Junon. Aurelius Victor, beneficiarius, érigea(ceci) à la suite d’un vœu.


Commentary:

Le dédicant est beneficiarius du légat de la légion ; voir pour cette charge A. v. Domaszewski, Rangordnung2, p. 38, 48.
