Text

D(is) [ M(anibus)]
M(arcus) Aur(elius) Au [ [. ] []]
Aug(ustalis) c [ ol(oniae) Nap(ocensis)(?) vix(it)]
an(nis) L A [ ur(elia) …]
5ca heṛ [ es[. ] []]
OR P [ [. ] []]
OPI(?) [ [. ] []]
[ [. ] [ ] [ ]].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

Aux dieux(Mânes). Marcus Aurelius Au…, Augustale de la colonie …, a vécu 0 ans. Aurelia …ca, héritière ….


Commentary:

L. 2 : Au [ gurinus], Au [ relianus] etc., voir H. Solin, O. Salomies, Repertorium, p. 297-299 ; l. 3, Torma : aug(ur) c [ oloniae Potaissae] ; Kerényi : Aug(ustalis) c [ ol(oniae) Pot(aissae)] ; Ardevan : Aug(ustalis) c [ ol(oniae) Pot(aissae)(?). L’abréviation AVG pour augur est évitée, afin de ne pas créer une confusion avec Aug(ustalis). D’autre part, l’Augustalité n’est pas attestée à Potaissa. Il s’agit donc plus probablement de la colonie de Napoca. L. 4- : le cognomen pourrait être [ Formi]ca, [ Mus]ca etc., voir H. Solin, O. Salomies, p. 429 ; l. -7, Torma : [ Cast ?] / or pe [ cunia sua] / ob m [ erita], cf. HE ; solutions peu probables.
