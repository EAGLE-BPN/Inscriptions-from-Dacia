Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Dé ou stéle funéraire,
Material: calcaire
Type of Inscription: Inscription funéraire
Execution Technique: sculptées
Letters: Points de séparation
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: À partir du règne d'Hadrien, en raison du nomen Aelius.

Bibliography:

  • Fr. Cumont, AEM 17, 1894, p. 18, no 4
  • CIL III 13762.

Text

[ M(anibus)]
Ael(ius, -ia) I [ [. ] []]
vix(it) an(nis) [ .. Ael(ius)]
Fero [ x vix(it) an(nis) ..]
5Ael(ius, -ia) Iu [ [. ] []]
vix(it) [ a]n(nis) [ [. ] []]
[ [. ] [ ] [ ]].

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

Aux dieux(Mânes). Aelius(Aelia ?) I… a vécu … ans.(Aelius) Ferox(a vécu … ans). Aelius(Aelia ?) Iu… a vécu … ans. ….
