Text

[I(ovi) O(ptimo) M(aximo)]
[c(ives) R(omani)] c(onsistentes)
[vi]c(o) clas(sicorum)
[cura(m)] ag(ente) Fl(avio)
5Vale[r]io ma[g(istro)
C[o]mm[odo]
[e]t Lael(iano) co(n)s(ulibus)

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

To the Jupiter the Good and the Greatest! The Roman citizens from sailors village set through the care of Flavius Valerius, the mayor, while Commodus and Laelianus were consuls.
