Source: UNIVERSITATEA BABES BOLYAI


Author: Rada Varga

Description

Object type: plaque
Material: marble
Type of Inscription: building inscription
Execution Technique: carved
Letters: 14.5
Provenance: Ulpia Traiana Sarmizegetusa
Findspot: forum vetus, Romania
Date: 106-117, during Trajan's reign

Bibliography:

  • I. Piso, Le vorum vetus de Sarmizegetusa, Bucuresti, 2006, p. 219-220

Text

[Imp(eratori) Caes(ari) Diu]i Ne[ruae fil(io) Neruae Traiano?]
[Aug(usto) Germani?]co [Dacico[---] ]


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

To the emperor Caesar Nerva Traianus, son of the divine Nerva, Augustus, Germanicus, Dacicus
