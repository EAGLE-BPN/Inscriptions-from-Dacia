Author: Ioan Piso

Description

Object type: Brique fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters:
Provenance: Tibiscum
Findspot: Jupa, Roumanie
Date: À partir du règne de Marc Aurèle, en raison de la séquence Marcus Aurelius.

Bibliography:

  • M. Moga, AMN 7, 1970, p. 145, fig. 5

Text

M(arci) A(urelii) Sy(ri).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Appartenant à Marcus Aurelius Syrus.


Commentary:

Estampille en cartouche rectangulaire, avec les lettres en relief; ligature MA. Le cognomen du personnage témoigne d'une origine syrienne. M. Aurelius Syrus est un propriétaire de domaine, dont il extrayait l'argile nécessaire aux tuiles.
