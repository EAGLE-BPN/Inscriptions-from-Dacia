Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Brique fragmentaire
Material: céramique
Type of Inscription: Estampille tégulaire
Execution Technique: imprimées avant cuisson
Letters:
Provenance: Brigetio
Findspot: Ó-Szöny, Hongrie
Date: À partir de l’arrivée, sous Domitien (81-96), de la legio I Adiutrix à Brigetio.

Bibliography:

  • CIL III 4655 k.

Text

Leg(ionis) I Adi(utricis)
Anto(nius) Mas(clus?).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Antonius Masclus(?), de la legio I Adiutrix.


Commentary:

Estampille faiblement imprimée dans la partie droite, en cartouche rectangulaire incomplet, ayant dans son intérieur une tabula ansata avec les anses inverses, entre les barres desquelles on distinge une sorte de palmette. Le texte est disposé sur deux registres séparés par une ligne; les bords et les lettres sont en relief. Th. Mommsen: LEG I ADIV / ANTOniNian. Pourtant, ce qu’on avait pris pour un V est en réalité la ligne intérieure verticale de la tabula ansata. Pour la l. 2, il ne s’agit pas de l’epithète impériale Antoniniana mais bien du nom de l’immunis qui dirigeait la briquetterie, un Anto(nius) Mas ou Mas(clus, -io); voir H. Solin, O. Salomies, Repertorium, p. 359.
