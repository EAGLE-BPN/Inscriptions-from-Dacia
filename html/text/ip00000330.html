Text

Ἀπο<λλι>νάρις Βάσσο[υ] εὐχὴν
[θε]ὸς Δολιχηνός.


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

Apollinaris, (fils de) Bassos, (a accompli) le vœu. Le dieu Dolichenos.


Commentary:

Le texte devrait commencer par le nom de la divinité au datif. On a pourtant préféré commencer par le nom du dédicant, suivi par la formule de conclusion. Ensuite, pour qu'il n'y ait aucun doute sur l'identité de la divinité, on y a ajouté le nom au nominatif. Pour la diffusion du nom Bassu dans le monde syrien voir H. Wuthnow, Die semitischen Menschennamen, p. 34 sq., 130 sq.; J. K. Stark, Personal Names, p. 77: BS’.
