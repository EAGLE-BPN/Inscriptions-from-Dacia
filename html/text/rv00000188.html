Source: UNIVERSITATEA BABES BOLYAI


Author: Rada Varga

Description

Object type: monument base
Material: limestone
Type of Inscription: epitaph
Execution Technique: carved
Letters: yes 9-9
Provenance: Napoca
Findspot: Gheorgheni, on the left bank of the river Becaș, Romania
Date: post Constitutio Antoniniana

Bibliography:

  • I. Piso, T. Tecar, Tibiscum 2013

Text

D(is) [M(anibus)]
Aurel(ius) M[axi-]
mus ve[t(eranus) ex]
dec(urione) vix(it)
5annis L[.]
Aurelii M[a-]
nes (centurio) Mụ-
saeus vex(illarius)
Maximus
10[F]austinus fili(i) pat(ri) pie(ntissimo).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

To the gods Manes of Aurelius Maximus, vetern, ex decurion, who lived 50 years. Aurelius Manes, centurion, Musaeus, vexillarius, Maximus Faustinus - the sons dedicated to their very pious father.
