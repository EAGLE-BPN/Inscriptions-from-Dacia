Indici
Source: UNIVERSITATEA BABES BOLYAI


Author: Florian Matei-Popescu

Description

Object type: Stele
Material: Sandstone
Type of Inscription: Funerary
Letters: Yes, in the first line 5-6
Provenance: Halmyris
Findspot: Reused in the Late Roman fort, as threshold of the phase II of the northern gate, Romania
Date: Probably second half of the second century AD

Bibliography:

  • AE 1991, 1388
  • Conrad, Grabstelen, p. 181-182, no. 212
  • M. Zahariade, C. Alexandrescu, Greek and Latin Inscriptions from Halmyris, 2011, 21

Text

D(is) M(anibus)
Fl(aviae) Titiae mat-
ri posuit Ael(ius)
Titianus filiu(s)
5bene meren-
ti vixit n(is) LX.

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

To the Gods of the Underworld! Aelius Titianus, the son, set for his well-deserving mother, Flavia Titia, who lived 70 years.
