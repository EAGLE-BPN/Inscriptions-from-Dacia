Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de statue
Material: tuf volcanique
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters: points de séparation
Provenance: Arcobadara
Findspot: Ilișua, Roumanie
Date: À partir du début du règne d’Hadrien, quand l’ala I Tungrorum Frontoniana est venue en Dacie.

Bibliography:

  • CIL III 788

Text

Epo [ nae]
ala I [ Tun]
gr(orum) Fro [ nton(iana)]
cui p [ rae(e)st]
5C(aius) Iul(ius) Api<c>i
anus praef(ectus)
eq(uitum)
[ v(otum) s(olvit)] l(ibens) m(erito).

Translation:

À Épone. L’ala I Tungrorum Frontoniana, à la tête de laquelle se trouve Caius Iulius Apicianus, préfet de cavaliers,(s’est acquittée de(son) vœu) de bon gré(et) à bon droit.


Commentary:

Une dédicace faite par une unité de cavalerie recrutée en Germanie à la déesse des chevaux(voir Keune, RE VI / 1(1907), 228 sqq.) est normale. L. -6, Torma, Nemeti, Gaiu, Zăgreanu : Apigianus ; cf. Mommsen : « APICIANVS malim » ; HD : Api<c>ianus. Le cognomen Apicianus est dérivé du nomen Apicius ; voir W. Schulze, Eigennamen, p. 292, 347, tandis qu’un nom Apigianus n’existe pas. Pour l’officier voir H. Devijver, PME I, J 22.
