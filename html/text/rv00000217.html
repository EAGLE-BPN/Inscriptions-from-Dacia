Source: UNIVERSITATEA BABES BOLYAI


Author: Rada Varga

Description

Object type: altar / statue base
Material: limestone
Type of Inscription: votive inscription
Execution Technique: carved
Letters: yes
Provenance:
Findspot: south of the village, Romania
Date: starting with the reign of Marcus Aurelius

Bibliography:

  • CIL III 7655

Text

Numin(i)
Asclep(ii)
p(ro) s(e) M(arcus) Aur(elius)
Vitalian(us)
5v(otum) s(olvit) l(ibens) m(erito).


File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

To the sacred force of Asclepius. For himself, Marcus Aurelius Vitalianus, fulfilled his promise with joy and for merit.
