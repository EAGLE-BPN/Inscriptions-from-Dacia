[... ][...]

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

See this file in GIT.



Translation:

... érigea à Jupiter Erusenus.


Commentary:

Manque la partie inférieure de l'inscription, qui contenait le nom du dédicant. La plupart des auteurs: Treuseno. I. I. Russu laissa ouverte la possibilité I(ovi) Eruseno. Il s'agit de la divinité galate Erusenus (Ἐρυσηνός). D'autres divinités d'Asie Mineure sont connues à Alburnus Maior dans AE 1944, 20 = SEG 25, 1971, 825 = IDR III/3, 432; AE 1944, 23 = SEG 25, 1971, 828 = IDR III/3, 399; AE 1944, 25 = SEG 25, 1971, 829 = IDR III/3, 398.
