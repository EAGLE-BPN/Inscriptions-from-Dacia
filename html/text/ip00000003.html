Source: UNIVERSITATEA BABES BOLYAI


Author: Ioan Piso

Description

Object type: Autel votif ou base de staute
Material: grès
Type of Inscription: Inscription votive
Execution Technique: sculptées
Letters:
Provenance: Potaissa
Findspot: Turda / Torda, Roumanie
Date: Probablement du IIIe siècle ap. J.-C., en raison du nomen Aurelius pas accompagné par praenomen.

Bibliography:

  • I. I. Russu, Z. Milea, Probleme de muzeografie, Cluj 1964, p. 31, no 2
  • M. Bărbulescu, AMN 14, 1977, p. 178, no 3

Text

Hercu
li Aur(elius)
Dubi(tatus)
v(otum) s(olvit) l(ibenter).

File sorgente prodotto da: EAGLE - Europeana Network of Ancient Greek and Latin Epigraphy

Translation:

À Hercule. Aurelius Dubitatus s’est acquitté de(son) vœu de bon gré.


Commentary:

L. 3 : on remarque l’abréviation insolite du cognomen.
